Button Component Demo

Button

사용자 액션을 트리거하는 기본 인터랙티브 요소입니다. data-variant로 5가지 스타일 변형, data-size로 sm / md / lg 크기를 지정합니다.

1. Primary

주요 액션(저장, 제출, 확인 등)에 사용하는 기본 버튼입니다. data-variant="primary"

2. Secondary

보조 액션(취소, 뒤로 등)에 사용하는 회색 버튼입니다. data-variant="secondary"

3. Outline

투명 배경에 테두리만 있는 버튼입니다. data-variant="outline"

4. Ghost

배경과 테두리가 없는 버튼입니다. 호버 시에만 배경이 나타납니다. data-variant="ghost"

5. Danger

삭제·초기화 등 파괴적 액션에 사용하는 경고 버튼입니다. data-variant="danger"

6. Icon

아이콘만 포함하는 정사각형 버튼입니다. aria-label로 기능을 반드시 명시하고, <i> 요소에 aria-hidden="true"를 추가합니다.

아이콘 + 텍스트 조합

7. Loading

비동기 작업 진행 중임을 나타냅니다. data-loading="true"와 aria-busy="true"를 함께 사용합니다.

변형별 로딩 상태

8. Disabled

비활성화된 버튼입니다. HTML disabled 속성을 추가하면 반투명 처리되고 클릭이 차단됩니다.
