Switch

ON/OFF 두 가지 상태를 전환하는 토글 스위치 컴포넌트입니다. data-switch 속성을 숨겨진 input[type="checkbox"]에 지정하고, 인접한 label[data-switch-label]의 ::before(트랙)와 ::after(노브) 가상 요소로 슬라이딩 토글을 렌더링합니다. :checked 상태에서 노브가 오른쪽으로 이동하고 트랙이 primary 색상으로 전환되며, data-size로 sm / md / lg 크기를 지정할 수 있습니다.

Default Switch

기본 스위치입니다. data-size 속성으로 sm(트랙 32×20px) / md(트랙 40×24px) / lg(트랙 52×32px) 크기를 지정합니다. 각 크기별로 OFF 상태와 ON(checked) 상태를 나란히 비교합니다.

Small (sm)

Medium (md) — 기본

Large (lg)

<!-- 기본 스위치: sm / md / lg 크기 변형 -->

<!-- sm OFF -->
<input type="checkbox" data-switch id="sw-sm-off" aria-label="Small Switch OFF" />
<label data-switch-label data-size="sm" for="sw-sm-off">OFF</label>

<!-- sm ON -->
<input type="checkbox" data-switch id="sw-sm-on" checked aria-label="Small Switch ON" />
<label data-switch-label data-size="sm" for="sw-sm-on">ON</label>

<!-- md(기본) OFF -->
<input type="checkbox" data-switch id="sw-md-off" aria-label="Medium Switch OFF" />
<label data-switch-label for="sw-md-off">OFF</label>

<!-- md(기본) ON -->
<input type="checkbox" data-switch id="sw-md-on" checked aria-label="Medium Switch ON" />
<label data-switch-label for="sw-md-on">ON</label>
/* 네이티브 checkbox 시각적 숨김 */
input[type="checkbox"][data-switch] {
  position: absolute;
  width: 1px; height: 1px;
  clip: rect(0, 0, 0, 0);
}

/* 트랙: pill 형태 배경 */
label[data-switch-label]::before {
  content: "";
  width: 40px; height: 24px;
  border-radius: 9999px;
  background-color: var(--color-border-hover);
  transition: background-color 0.3s cubic-bezier(0.4,0,0.2,1);
}

/* 노브: 슬라이딩 원형 */
label[data-switch-label]::after {
  content: "";
  position: absolute;
  left: 0; top: 50%;
  width: 18px; height: 18px;
  background-color: #ffffff;
  border-radius: 50%;
  transform: translateX(3px) translateY(-50%);
  transition: transform 0.3s cubic-bezier(0.4,0,0.2,1);
}

/* ON 상태 */
input[type="checkbox"][data-switch]:checked + label[data-switch-label]::before {
  background-color: var(--color-primary);
}
input[type="checkbox"][data-switch]:checked + label[data-switch-label]::after {
  transform: translateX(19px) translateY(-50%);
}
// 스위치 상태 변경 감지
const switches = document.querySelectorAll('[data-switch]');
switches.forEach((sw) => {
  sw.addEventListener('change', (e) => {
    const state = e.target.checked ? 'ON' : 'OFF';
    console.log(`Switch 상태: ${state}`);
  });
});

With Label & Helper

스위치 앞에 설명 레이블을, 아래에 헬퍼 텍스트를 배치하는 패턴입니다. aria-describedby로 헬퍼 텍스트 ID를 연결하면 스크린리더가 스위치 목적과 부가 정보를 함께 읽습니다. role="switch"와 aria-checked로 접근성 시멘틱을 강화합니다.

마케팅 이메일 및 서비스 업데이트를 받습니다.

시스템 설정에 관계없이 항상 다크 테마를 사용합니다.

개인정보 처리방침 동의 *

서비스 이용을 위해 필수로 동의해야 합니다. 

<!-- role="switch" + aria-checked로 접근성 시멘틱 명시 -->
<div>
  <input
    type="checkbox"
    data-switch
    id="sw-notify"
    role="switch"
    aria-checked="false"
    aria-describedby="sw-notify-helper"
  />
  <label data-switch-label for="sw-notify">
    이메일 알림 수신
  </label>
</div>
<p class="switch-helper" id="sw-notify-helper">
  마케팅 이메일 및 서비스 업데이트를 받습니다.
</p>
/* 헬퍼 텍스트 스타일 */
.switch-helper {
  font-size: var(--font-size-sm);
  color: var(--color-text-secondary);
  margin: 0;
  margin-top: var(--spacing-xs);
}
// aria-checked 동적 업데이트
const sw = document.querySelector('#sw-notify');
sw.addEventListener('change', (e) => {
  sw.setAttribute('aria-checked', e.target.checked ? 'true' : 'false');
});

Error State

data-error="true"와 aria-invalid="true"를 함께 지정하면 트랙이 danger 계열 색상으로 전환됩니다. 에러 메시지는 role="alert"를 사용해 스크린리더가 즉시 읽습니다.

서비스 이용을 위해 필수로 동의해야 합니다.

이 기능을 활성화하면 데이터가 즉시 삭제됩니다. 

<!-- 에러: data-error="true" + aria-invalid="true" -->
<div>
  <input
    type="checkbox"
    data-switch
    id="sw-err-off"
    data-error="true"
    aria-invalid="true"
    aria-describedby="sw-err-msg"
    role="switch"
    aria-checked="false"
  />
  <label data-switch-label for="sw-err-off">
    서비스 이용 동의
  </label>
</div>
<p class="switch-error-msg" id="sw-err-msg" role="alert">
  서비스 이용을 위해 필수로 동의해야 합니다.
</p>
/* 에러 트랙 */
input[type="checkbox"][data-switch][data-error="true"]
+ label[data-switch-label]::before {
  background-color: color-mix(
    in srgb, var(--color-danger) 25%, var(--color-border-hover)
  );
}
/* 에러 + ON */
input[type="checkbox"][data-switch][data-error="true"]:checked
+ label[data-switch-label]::before {
  background-color: var(--color-danger);
}
// 에러 상태 동적 적용
function setError(input, hasError) {
  input.setAttribute('data-error', hasError ? 'true' : 'false');
  input.setAttribute('aria-invalid', hasError ? 'true' : 'false');
}

Disabled Switch

disabled 속성으로 스위치를 비활성화합니다. 시각적으로 반투명(opacity 0.5)하게 표시되며 커서가 not-allowed로 변경됩니다. 비활성화된 OFF와 ON 상태 모두 표시하여 읽기 전용 상태를 나타낼 수 있습니다. 

<!-- 비활성 OFF -->
<input
  type="checkbox"
  data-switch
  id="sw-dis-off"
  disabled
  aria-label="비활성화 스위치 OFF"
/>
<label data-switch-label for="sw-dis-off">비활성화 (OFF)</label>

<!-- 비활성 ON (읽기 전용) -->
<input
  type="checkbox"
  data-switch
  id="sw-dis-on"
  disabled
  checked
  aria-label="비활성화 스위치 ON"
/>
<label data-switch-label for="sw-dis-on">비활성화 (ON)</label>
/* 비활성화: opacity + not-allowed */
input[type="checkbox"][data-switch]:disabled
+ label[data-switch-label] {
  opacity: 0.5;
  cursor: not-allowed;
  pointer-events: none;
}
// 동적 비활성화 토글
const sw = document.querySelector('#sw-dis-off');
sw.disabled = true; // 비활성화
sw.disabled = false; // 재활성화

Switch with Description

좌측에 제목과 설명 텍스트, 우측에 스위치를 배치하는 일반적인 설정 화면 패턴입니다. .switch-row 클래스로 justify-content: space-between 레이아웃을 구현합니다. aria-labelledby로 제목 ID를 연결하면 스크린리더가 스위치 기능을 명확하게 전달합니다.

푸시 알림

앱 업데이트 및 새 메시지를 실시간으로 받습니다.

자동 저장

5분마다 작업 내용을 자동으로 저장합니다.

위치 정보 공유

현재 권한 없음. 설정에서 위치 접근을 허용하세요.

<div class="switch-row">
  <div class="switch-row__content">
    <span class="switch-row__title" id="sw-push-title">
      푸시 알림
    </span>
    <p class="switch-row__desc">
      앱 업데이트 및 새 메시지를 실시간으로 받습니다.
    </p>
  </div>
  <div>
    <input
      type="checkbox"
      data-switch
      id="sw-push"
      role="switch"
      aria-checked="true"
      aria-labelledby="sw-push-title"
      checked
    />
    <label data-switch-label for="sw-push">
      <span class="sr-only">푸시 알림 켜기/끄기</span>
    </label>
  </div>
</div>
/* 좌우 배치 컨테이너 */
.switch-row {
  display: flex;
  align-items: center;
  justify-content: space-between;
  gap: var(--spacing-md);
  padding: var(--spacing-sm) 0;
}

.switch-row__content {
  flex: 1;
  min-width: 0;
}

.switch-row__title {
  font-weight: 500;
  color: var(--color-text);
}

.switch-row__desc {
  font-size: var(--font-size-sm);
  color: var(--color-text-secondary);
  margin: 0;
}
// aria-checked 동적 업데이트
const sw = document.querySelector('#sw-push');
sw.addEventListener('change', (e) => {
  sw.setAttribute('aria-checked', e.target.checked ? 'true' : 'false');
});

Multiple Switches

관련 스위치 여러 개를 그룹으로 묶어 설정 목록처럼 표시하는 패턴입니다. .switch-list + .switch-list__item으로 테두리와 구분선을 구성하고, 각 항목은 .switch-row 레이아웃을 재사용합니다.

// 다중 스위치 상태 수집
function getSettings() {
  const settings = {};
  document.querySelectorAll('.switch-list [data-switch]')
    .forEach((sw) => { settings[sw.id] = sw.checked; });
  return settings;
}

// aria-checked 동기화
document.querySelectorAll('[data-switch]').forEach((sw) => {
  sw.addEventListener('change', (e) => {
    e.target.setAttribute(
      'aria-checked',
      e.target.checked ? 'true' : 'false'
    );
  });
});
