Dropdown

버튼을 클릭하면 메뉴 목록이 펼쳐지는 컴포넌트입니다. WAI-ARIA menu/menuitem 패턴을 준수하며 data-component="dropdown"으로 컨테이너를 지정합니다. data-dropdown-trigger 버튼과 role="menu" 목록, role="menuitem" 항목으로 구성됩니다. JavaScript 없이도 목록이 펼쳐진 상태로 접근 가능합니다.

Default Dropdown

기본 드롭다운입니다. 버튼을 클릭하면 메뉴가 펼쳐집니다. JavaScript 없이도 목록이 펼쳐진 상태로 접근할 수 있습니다. 

<div class="dropdown" data-component="dropdown">
  <button
    class="dropdown__trigger"
    type="button"
    data-dropdown-trigger
    aria-haspopup="true"
    aria-expanded="false"
    aria-controls="menu-1"
  >
    메뉴
    <i data-lucide="chevron-down" class="dropdown__chevron" aria-hidden="true"></i>
  </button>

  <ul id="menu-1" class="dropdown__menu" role="menu">
    <li role="none">
      <button class="dropdown__item" type="button" role="menuitem" tabindex="-1">
        항목 1
      </button>
    </li>
    <li role="none">
      <button class="dropdown__item" type="button" role="menuitem" tabindex="-1">
        항목 2
      </button>
    </li>
    <li role="none">
      <button class="dropdown__item" type="button" role="menuitem" tabindex="-1">
        항목 3
      </button>
    </li>
  </ul>
</div>
/* 드롭다운 래퍼: 포지셔닝 기준점 */
.dropdown {
  position: relative;
  display: inline-block;
}

/* 메뉴 목록 */
.dropdown__menu {
  position: absolute;
  top: calc(100% + 4px);
  left: 0;
  z-index: 200;
  min-width: 180px;
  background-color: var(--color-bg);
  border: 1px solid var(--color-border);
  border-radius: var(--radius-md);
  box-shadow: 0 4px 6px -1px rgb(0 0 0 / 0.1);
  list-style: none;
  padding: 0.25rem 0;
}

/* 트리거 아이콘: 열림 상태에서 180도 회전 */
[data-dropdown-trigger][aria-expanded="true"] .dropdown__chevron {
  transform: rotate(180deg);
}
// DOMContentLoaded 시 자동 초기화
document.addEventListener('DOMContentLoaded', () => {
  Dropdown.init(); // 페이지의 모든 [data-component="dropdown"] 초기화
});

// 특정 컨테이너 범위만 초기화:
Dropdown.init(document.querySelector('.my-container'));

With Icons

각 메뉴 항목에 아이콘을 추가하여 시각적 컨텍스트를 제공하는 패턴입니다. 아이콘에는 aria-hidden="true"를 적용하여 스크린 리더가 중복 읽기를 방지합니다. 

<ul id="menu-icons" class="dropdown__menu" role="menu">
  <li role="none">
    <button class="dropdown__item" type="button" role="menuitem" tabindex="-1">
      <!-- aria-hidden="true": 아이콘은 스크린 리더가 건너뜀 -->
      <i data-lucide="user" class="dropdown__item-icon" aria-hidden="true"></i>
      내 프로필
    </button>
  </li>
  <li role="none">
    <button class="dropdown__item" type="button" role="menuitem" tabindex="-1">
      <i data-lucide="settings" class="dropdown__item-icon" aria-hidden="true"></i>
      설정
    </button>
  </li>
</ul>
/* 아이콘 크기 고정: 텍스트 길이에 관계없이 일정한 너비 */
.dropdown__item-icon {
  width: 16px;
  height: 16px;
  flex-shrink: 0;
  color: var(--color-text-secondary);
}

.dropdown__item {
  display: flex;
  align-items: center;
  gap: var(--spacing-sm);
}
// 아이콘 항목도 일반 menuitem과 동일하게 동작
// 별도 JS 설정 불필요
// 스크린 리더: 아이콘(aria-hidden) 건너뛰고 텍스트만 읽음

Divider & Disabled

구분선(.dropdown__divider)으로 관련 항목을 그룹화하고, aria-disabled="true"로 비활성 항목을 처리합니다. 비활성 항목은 키보드 탐색에서 건너뜁니다. 

<ul id="menu-divider" class="dropdown__menu" role="menu">
  <li role="none">
    <button class="dropdown__item" type="button" role="menuitem" tabindex="-1">
      새 파일
    </button>
  </li>

  <!-- 구분선: role="separator" -->
  <li role="separator" class="dropdown__divider" aria-hidden="true"></li>

  <!-- aria-disabled="true": 키보드 탐색에서 자동 건너뜀 -->
  <li role="none">
    <button
      class="dropdown__item"
      type="button"
      role="menuitem"
      tabindex="-1"
      aria-disabled="true"
    >
      다른 이름으로 저장
    </button>
  </li>
</ul>
/* 구분선 */
.dropdown__divider {
  height: 1px;
  background-color: var(--color-border);
  margin: 0.25rem 0;
}

/* 비활성 항목 */
.dropdown__item[aria-disabled="true"] {
  color: var(--color-text-disabled, #94a3b8);
  cursor: not-allowed;
  pointer-events: none;
  opacity: 0.5;
}
// aria-disabled="true" 항목은 _getEnabledItems()에서 자동 제외
// 키보드 Arrow Down/Up 탐색 시 건너뜀
// 스크린 리더: "비활성" 상태 알림

// 비활성 항목 포함된 enabled items 필터링:
const enabled = Array.from(menu.querySelectorAll('[role="menuitem"]'))
  .filter(item => item.getAttribute('aria-disabled') !== 'true');

Danger Action

삭제와 같이 위험한 액션은 .dropdown__item--danger 클래스로 빨간색 강조합니다. .dropdown--right 클래스로 메뉴를 우측 정렬할 수 있습니다. 

<!-- .dropdown--right: 우측 정렬 -->
<div class="dropdown dropdown--right" data-component="dropdown">
  <button class="dropdown__trigger" type="button"
    data-dropdown-trigger aria-haspopup="true"
    aria-expanded="false" aria-controls="menu-danger">
    더보기
  </button>

  <ul id="menu-danger" class="dropdown__menu" role="menu">
    <li role="none">
      <button class="dropdown__item" type="button" role="menuitem" tabindex="-1">
        편집
      </button>
    </li>
    <li role="separator" class="dropdown__divider" aria-hidden="true"></li>
    <!-- .dropdown__item--danger: 위험 액션 빨간색 강조 -->
    <li role="none">
      <button
        class="dropdown__item dropdown__item--danger"
        type="button" role="menuitem" tabindex="-1"
      >
        <i data-lucide="trash-2" class="dropdown__item-icon" aria-hidden="true"></i>
        삭제
      </button>
    </li>
  </ul>
</div>
/* 우측 정렬: 버튼 오른쪽 끝에 메뉴 정렬 */
.dropdown--right .dropdown__menu {
  left: auto;
  right: 0;
}

/* 위험 액션: 빨간색 텍스트 */
.dropdown__item--danger {
  color: var(--color-danger, #ef4444);
}

/* 호버/포커스 시 빨간 배경 */
.dropdown__item--danger:hover,
.dropdown__item--danger[data-focused="true"] {
  background-color: #fef2f2;
  color: var(--color-danger, #ef4444);
}
// 위험 액션 클릭 전 확인 다이얼로그 예시:
document.querySelector('.dropdown__item--danger')
  .addEventListener('click', () => {
    if (confirm('정말 삭제하시겠습니까?')) {
      // 삭제 로직 실행
    }
  });
