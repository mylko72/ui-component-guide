Alert

Alert 컴포넌트는 사용자에게 중요한 메시지, 경고, 오류 정보를 전달합니다. 4가지 타입(Info, Success, Warning, Error)을 지원하며, role="alert" ARIA 패턴을 사용하여 스크린리더 사용자에게 즉시 공지됩니다. 닫기 버튼으로 수동 해제 가능합니다.

4가지 Alert 타입

Info, Success, Warning, Error 4가지 상태의 Alert 컴포넌트입니다. 각 Alert은 해당하는 의미론적 역할을 수행합니다.

정보

이것은 정보 알림입니다. 사용자에게 도움이 될 만한 추가 정보를 제공합니다.

성공

작업이 성공적으로 완료되었습니다. 사용자의 요청이 처리되었습니다.

경고

주의가 필요합니다. 진행하기 전에 다시 한번 확인하시기 바랍니다.

오류

오류가 발생했습니다. 잠시 후 다시 시도하거나 관리자에게 문의하세요.

<!-- role="alert": 스크린 리더에 즉시 공지 -->
<div role="alert" class="alert alert--info">
  <div class="alert-icon">
    <i data-lucide="info" aria-hidden="true"></i>
  </div>
  <div class="alert-content">
    <strong class="alert-title">정보</strong>
    <p class="alert-message">이것은 정보 알림입니다.</p>
  </div>
  <button type="button" class="alert-close" data-alert-close aria-label="닫기">
    <i data-lucide="x" aria-hidden="true"></i>
  </button>
</div>

<!-- 타입: --info / --success / --warning / --error -->
<div role="alert" class="alert alert--success">...</div>
<div role="alert" class="alert alert--warning">...</div>
<div role="alert" class="alert alert--error">...</div>

// Alert 컴포넌트: 닫기 버튼 기능
class Alert {
  static init(container = document) {
    container.querySelectorAll('[data-alert]').forEach(el => {
      // 중복 초기화 방지
      if (el.dataset.initialized) return;
      el.dataset.initialized = 'true';

      // 닫기 버튼 이벤트 등록
      const closeBtn = el.querySelector('[data-alert-close]');
      if (closeBtn) {
        closeBtn.addEventListener('click', () => {
          el.remove();
        });
      }
    });
  }
}

// 페이지 로드 시 자동 초기화
document.addEventListener('DOMContentLoaded', () => {
  Alert.init();
});
