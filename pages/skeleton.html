Skeleton

콘텐츠가 로딩되는 동안 실제 레이아웃을 미리 표시하는 플레이스홀더 컴포넌트입니다. .skeleton__line으로 텍스트 줄을, .skeleton-card로 카드 레이아웃을, .skeleton-table로 테이블 행을, .skeleton-avatar로 아바타를 표현합니다. JavaScript 없이 CSS @keyframes 시머 애니메이션만으로 동작하며, prefers-reduced-motion을 완전히 지원합니다.

텍스트 Skeleton

가장 기본적인 텍스트 로딩 상태입니다. .skeleton 컨테이너 안에 .skeleton__line을 배치하여 여러 줄의 텍스트가 로딩 중임을 표현합니다. .skeleton__line--short로 문단 마지막 줄의 짧은 끝을 자연스럽게 표현합니다. 스크린 리더에 aria-busy="true"와 aria-label로 로딩 상태를 알립니다. 

<!-- 텍스트 스켈레톤: 여러 줄 로딩 표현 -->
<div
  class="skeleton"
  role="status"
  aria-busy="true"
  aria-label="텍스트 로딩 중"
>
  <!-- 제목 줄: 두꺼운 높이 -->
  <div class="skeleton__line skeleton__line--title"></div>

  <!-- 본문 줄: 전체 너비 -->
  <div class="skeleton__line"></div>
  <div class="skeleton__line"></div>

  <!-- 마지막 줄: 60% 너비로 문단 끝 표현 -->
  <div class="skeleton__line skeleton__line--short"></div>
</div>
/* 베이스 컨테이너 */
.skeleton {
  display: flex;
  flex-direction: column;
  gap: var(--spacing-sm);
}

/* 로딩 바 */
.skeleton__line {
  height: 12px;
  width: 100%;
  border-radius: var(--radius-sm);
  display: block;

  /* 시머 그래디언트 */
  background: linear-gradient(
    90deg,
    var(--color-bg-secondary) 0%,
    var(--color-border)       25%,
    var(--color-bg-secondary) 50%,
    var(--color-border)       75%,
    var(--color-bg-secondary) 100%
  );
  background-size: 2000px 100%;
  animation: skeleton-shimmer 1.8s linear infinite;
}

/* 너비 변형 */
.skeleton__line--short  { width: 60%; }
.skeleton__line--medium { width: 80%; }
.skeleton__line--title  { height: 20px; width: 70%; }

/* 키프레임 */
@keyframes skeleton-shimmer {
  0%   { background-position: -1000px 0; }
  100% { background-position:  1000px 0; }
}

카드 Skeleton

Card 컴포넌트의 로딩 상태를 표현합니다. .skeleton-card__header는 이미지 커버 영역(200px)을, .skeleton-card__body 안의 .skeleton__line들은 제목과 설명 텍스트의 로딩을 나타냅니다. 반응형 그리드(.skeleton-card-grid)로 여러 카드를 배치합니다. 

<!-- 카드 스켈레톤 그리드 -->
<div class="skeleton-card-grid">

  <!-- 카드 스켈레톤 단일 아이템 -->
  <div
    class="skeleton-card"
    role="status"
    aria-busy="true"
    aria-label="카드 콘텐츠 로딩 중"
  >
    <!-- 이미지 커버 영역 (200px) -->
    <div class="skeleton-card__header" aria-hidden="true"></div>

    <!-- 텍스트 콘텐츠 영역 -->
    <div class="skeleton-card__body">
      <div class="skeleton__line skeleton__line--title"></div>
      <div class="skeleton__line"></div>
      <div class="skeleton__line skeleton__line--short"></div>
    </div>
  </div>

</div>
/* 카드 스켈레톤 */
.skeleton-card {
  border: 1px solid var(--color-border);
  border-radius: var(--radius-md);
  background-color: var(--color-bg);
  box-shadow: var(--shadow-sm);
  overflow: hidden;
}

/* 이미지 커버 영역 */
.skeleton-card__header {
  height: 200px;
  background: linear-gradient(
    90deg,
    var(--color-bg-secondary) 0%,
    var(--color-border)       25%,
    var(--color-bg-secondary) 50%,
    var(--color-border)       75%,
    var(--color-bg-secondary) 100%
  );
  background-size: 2000px 100%;
  animation: skeleton-shimmer 1.8s linear infinite;
}

/* 본문 영역 */
.skeleton-card__body {
  padding: var(--spacing-md);
  display: flex;
  flex-direction: column;
  gap: var(--spacing-sm);
}

/* 반응형 그리드 */
.skeleton-card-grid {
  display: grid;
  gap: var(--spacing-md);
  grid-template-columns: 1fr;
}
@media (min-width: 640px) {
  .skeleton-card-grid { grid-template-columns: repeat(2, 1fr); }
}
@media (min-width: 1024px) {
  .skeleton-card-grid { grid-template-columns: repeat(3, 1fr); }
}

테이블 행 Skeleton

테이블 형태의 데이터가 로딩 중임을 표현합니다. table.skeleton-table 구조 안에서 각 행(tr.skeleton-row)과 셀(td.skeleton-cell) 안에 .skeleton__line을 배치합니다. 열 너비에 따라 로딩 바의 너비를 다르게 조정할 수 있습니다.

# 이름 이메일 상태 액션 
<!-- 테이블 스켈레톤 -->
<table
  class="skeleton-table"
  role="status"
  aria-busy="true"
  aria-label="테이블 데이터 로딩 중"
>
  <tbody>

    <!-- 스켈레톤 행 (4행 반복) -->
    <tr class="skeleton-row">

      <!-- 번호 셀 -->
      <td class="skeleton-cell">
        <div class="skeleton__line" style="width:20px;"></div>
      </td>

      <!-- 이름 셀 -->
      <td class="skeleton-cell">
        <div class="skeleton__line" style="width:100px;"></div>
      </td>

      <!-- 이메일 셀 -->
      <td class="skeleton-cell">
        <div class="skeleton__line" style="width:160px;"></div>
      </td>

      <!-- 배지형 상태 -->
      <td class="skeleton-cell">
        <div class="skeleton__line"
          style="width:60px;border-radius:9999px;"></div>
      </td>

      <!-- 버튼형 액션 -->
      <td class="skeleton-cell">
        <div class="skeleton__line"
          style="width:56px;height:28px;
                 border-radius:var(--radius-md);
                 margin-left:auto;"></div>
      </td>

    </tr>

  </tbody>
</table>
/* 테이블 스켈레톤 */
.skeleton-table {
  width: 100%;
  border-collapse: collapse;
}

/* 행 하단 구분선 */
.skeleton-row {
  border-bottom: 1px solid var(--color-border);
}

.skeleton-row:last-child {
  border-bottom: none;
}

/* 셀 여백 */
.skeleton-cell {
  padding: var(--spacing-md) var(--spacing-sm);
  vertical-align: middle;
}

/* 모바일 조정 */
@media (max-width: 480px) {
  .skeleton-cell {
    padding: var(--spacing-sm) var(--spacing-xs);
  }
}

아바타 Skeleton

사용자 프로필이나 코멘트 목록에서 자주 사용되는 아바타 + 텍스트 조합의 로딩 상태입니다. .skeleton-avatar__image로 원형(48px) 아바타를, .skeleton-avatar__content로 이름과 설명 텍스트 줄을 표현합니다. .skeleton-avatar-list로 여러 아바타 행을 일정 간격으로 나열합니다. 

<!-- 아바타 스켈레톤 목록 -->
<div
  class="skeleton-avatar-list"
  role="status"
  aria-busy="true"
  aria-label="사용자 목록 로딩 중"
>

  <!-- 아바타 행 (여러 번 반복) -->
  <div class="skeleton-avatar">

    <!-- 원형 이미지 영역 (48px) -->
    <div class="skeleton-avatar__image" aria-hidden="true"></div>

    <!-- 텍스트 영역 -->
    <div class="skeleton-avatar__content">
      <div class="skeleton__line" style="width:120px;"></div>
      <div class="skeleton__line skeleton__line--short"
           style="width:80px;"></div>
    </div>

  </div>

</div>

<!-- 코멘트형: 작은 아바타 + 긴 텍스트 -->
<div class="skeleton-avatar">
  <div class="skeleton-avatar__image"
       style="width:40px;height:40px;" aria-hidden="true"></div>
  <div class="skeleton-avatar__content">
    <div class="skeleton__line" style="width:90px;"></div>
    <div class="skeleton__line"></div>
    <div class="skeleton__line skeleton__line--short"></div>
  </div>
</div>
/* 아바타 스켈레톤 */
.skeleton-avatar {
  display: flex;
  align-items: flex-start;
  gap: var(--spacing-sm);
  padding: var(--spacing-xs) 0;
}

/* 원형 아바타 이미지 */
.skeleton-avatar__image {
  width: 48px;
  height: 48px;
  border-radius: 50%;
  flex-shrink: 0;
  background: linear-gradient(
    90deg,
    var(--color-bg-secondary) 0%,
    var(--color-border)       25%,
    var(--color-bg-secondary) 50%,
    var(--color-border)       75%,
    var(--color-bg-secondary) 100%
  );
  background-size: 2000px 100%;
  animation: skeleton-shimmer 1.8s linear infinite;
}

/* 텍스트 콘텐츠 영역 */
.skeleton-avatar__content {
  flex: 1;
  min-width: 0;
  display: flex;
  flex-direction: column;
  gap: var(--spacing-xs);
  padding-top: var(--spacing-xs);
}

/* 목록 래퍼 */
.skeleton-avatar-list {
  display: flex;
  flex-direction: column;
  gap: var(--spacing-xs);
}

/* 모션 감소: 시머 → 펄스 전환 */
@media (prefers-reduced-motion: reduce) {
  .skeleton-avatar__image {
    animation: skeleton-pulse 2s ease-in-out infinite;
    background: var(--color-bg-secondary);
  }
}
