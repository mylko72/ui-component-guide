Radio

여러 옵션 중 하나를 선택하는 라디오 버튼 컴포넌트입니다. data-radio 속성을 숨겨진 input[type="radio"]에 지정하고, 인접한 label[data-radio-label]의 ::before/::after 가상 요소로 커스텀 원형과 내부 점을 렌더링합니다. 동일한 name 속성을 가진 라디오끼리 그룹을 형성하며, data-size로 sm / md / lg 크기를 지정하고 에러 상태, 비활성화를 지원합니다.

Default Radio

기본 라디오 버튼입니다. data-size 속성으로 sm(32px) / md(44px) / lg(52px) 크기를 지정합니다. 동일한 name을 가진 라디오끼리 그룹을 이루며 하나만 선택됩니다.

Small (sm)

Medium (md) — 기본

Large (lg)

<!-- 기본 라디오 그룹 (크기: sm / md / lg) -->
<!-- 동일 name 속성으로 그룹 형성: 하나만 선택 가능 -->

<!-- sm 크기 그룹 -->
<div class="radio-group">
  <div>
    <input type="radio" data-radio id="r-sm-1" name="r-sm" value="opt1" />
    <label data-radio-label data-size="sm" for="r-sm-1">옵션 1</label>
  </div>
  <div>
    <input type="radio" data-radio id="r-sm-2" name="r-sm" value="opt2" checked />
    <label data-radio-label data-size="sm" for="r-sm-2">옵션 2 (선택됨)</label>
  </div>
</div>

<!-- md(기본) 크기 그룹 -->
<div class="radio-group">
  <div>
    <input type="radio" data-radio id="r-md-1" name="r-md" value="opt1" />
    <label data-radio-label for="r-md-1">옵션 1</label>
  </div>
  <div>
    <input type="radio" data-radio id="r-md-2" name="r-md" value="opt2" checked />
    <label data-radio-label for="r-md-2">옵션 2 (선택됨)</label>
  </div>
</div>
/* 네이티브 라디오 시각적 숨김 (접근성 트리 유지) */
input[type="radio"][data-radio] {
  position: absolute;
  width: 1px; height: 1px;
  clip: rect(0, 0, 0, 0);
}

/* ::before: 원형 외테두리 */
label[data-radio-label]::before {
  content: "";
  width: 18px; height: 18px;
  border: 2px solid var(--color-border-hover);
  border-radius: 50%; /* 핵심: 원형 */
  background-color: var(--color-bg);
}

/* ::after: 내부 점 */
label[data-radio-label]::after {
  content: "";
  position: absolute;
  left: 0; top: 50%;
  width: 6px; height: 6px;
  background-color: var(--color-primary);
  border-radius: 50%;
  transform: translateX(6px) translateY(-50%);
  opacity: 0;
}

/* 선택 상태 */
input[type="radio"][data-radio]:checked + label[data-radio-label]::before {
  border-color: var(--color-primary);
}
input[type="radio"][data-radio]:checked + label[data-radio-label]::after {
  opacity: 1;
}
// 라디오 선택 변경 감지 예시
const radios = document.querySelectorAll('[data-radio]');
radios.forEach((radio) => {
  radio.addEventListener('change', (e) => {
    if (e.target.checked) {
      console.log('선택된 값:', e.target.value);
    }
  });
});

With Label & Helper

라디오 그룹은 반드시 fieldset + legend로 묶어 스크린리더가 그룹 맥락을 올바르게 전달하도록 합니다. aria-describedby로 헬퍼 텍스트를 연결합니다.

배송 방법 *

배송 방법에 따라 추가 요금이 부과될 수 있습니다.

구독 플랜

플랜은 언제든지 변경하실 수 있습니다. 

<!-- 라디오 그룹: fieldset + legend (접근성 필수 구조) -->
<fieldset style="border: none; padding: 0;">
  <legend class="radio-group-label">
    배송 방법
    <span class="radio-required" aria-hidden="true">*</span>
  </legend>
  <div class="radio-group" aria-describedby="shipping-helper">
    <div>
      <input type="radio" data-radio
        id="ship-std" name="shipping" value="standard"
        checked aria-describedby="shipping-helper" />
      <label data-radio-label for="ship-std">일반 배송 (3~5일)</label>
    </div>
    <div>
      <input type="radio" data-radio
        id="ship-exp" name="shipping" value="express"
        aria-describedby="shipping-helper" />
      <label data-radio-label for="ship-exp">빠른 배송 (1~2일)</label>
    </div>
    <div>
      <input type="radio" data-radio
        id="ship-same" name="shipping" value="same-day"
        aria-describedby="shipping-helper" />
      <label data-radio-label for="ship-same">당일 배송</label>
    </div>
  </div>
  <p class="radio-helper" id="shipping-helper">
    배송 방법에 따라 추가 요금이 부과될 수 있습니다.
  </p>
</fieldset>
/* 라디오 그룹 레이블 */
.radio-group-label {
  display: block;
  font-size: var(--font-size-sm);
  font-weight: 500;
  color: var(--color-text);
  margin-bottom: var(--spacing-xs);
}

/* 필수 표시 */
.radio-required {
  color: var(--color-danger);
  margin-left: 1px;
}

/* 수직 그룹 */
.radio-group {
  display: flex;
  flex-direction: column;
  gap: var(--spacing-sm);
}

/* 헬퍼 텍스트 */
.radio-helper {
  font-size: var(--font-size-sm);
  color: var(--color-text-secondary);
  margin: 0;
  margin-top: var(--spacing-xs);
}
// 라디오 그룹 선택 값 수집 예시
const shippingRadios = document.querySelectorAll('[name="shipping"]');
shippingRadios.forEach((radio) => {
  radio.addEventListener('change', (e) => {
    console.log('선택된 배송 방법:', e.target.value);
  });
});

Error State

필수 선택 미완료 등 유효성 검사 실패 시 data-error="true"와 aria-invalid="true", 에러 메시지를 함께 제공합니다. 그룹 내 모든 라디오에 data-error를 적용합니다.

결제 방법 *

결제 방법을 선택해 주세요.

나이 범위 *

이 서비스는 18세 이상만 이용 가능합니다. 

<!-- 에러 상태: 모든 라디오에 data-error + aria-invalid -->
<fieldset style="border: none; padding: 0;">
  <legend class="radio-group-label">
    결제 방법
    <span class="radio-required" aria-hidden="true">*</span>
  </legend>
  <div class="radio-group"
    aria-describedby="payment-error"
    aria-invalid="true"
  >
    <div>
      <input type="radio" data-radio
        data-error="true"
        id="pay-card" name="payment" value="card"
        aria-invalid="true" aria-describedby="payment-error" />
      <label data-radio-label for="pay-card">신용카드</label>
    </div>
    <div>
      <input type="radio" data-radio
        data-error="true"
        id="pay-bank" name="payment" value="bank"
        aria-invalid="true" aria-describedby="payment-error" />
      <label data-radio-label for="pay-bank">계좌이체</label>
    </div>
  </div>
  <p class="radio-error-msg" id="payment-error" role="alert">
    <i data-lucide="alert-circle" aria-hidden="true"></i>
    결제 방법을 선택해 주세요.
  </p>
</fieldset>
/* 에러: 빨간 테두리 + 연한 배경 */
input[type="radio"][data-radio][data-error="true"]
  + label[data-radio-label]::before {
  border-color: var(--color-danger);
  background-color: color-mix(in srgb, var(--color-danger) 6%, var(--color-bg));
}

/* 에러 + 선택: 내부 점 danger 색상 */
input[type="radio"][data-radio][data-error="true"]:checked
  + label[data-radio-label]::after {
  background-color: var(--color-danger);
  opacity: 1;
}

/* 에러 메시지 */
.radio-error-msg {
  display: flex;
  align-items: center;
  gap: 4px;
  font-size: var(--font-size-sm);
  color: var(--color-danger);
  margin-top: var(--spacing-xs);
}
// 폼 제출 시 라디오 그룹 유효성 검사
const form = document.querySelector('form');
form.addEventListener('submit', (e) => {
  const radios = document.querySelectorAll('[name="payment"]');
  const selected = [...radios].find((r) => r.checked);
  const errorMsg = document.querySelector('#payment-error');

  if (!selected) {
    e.preventDefault();
    radios.forEach((r) => {
      r.dataset.error = 'true';
      r.setAttribute('aria-invalid', 'true');
    });
    errorMsg.hidden = false;
    radios[0].focus();
  } else {
    radios.forEach((r) => {
      delete r.dataset.error;
      r.setAttribute('aria-invalid', 'false');
    });
    errorMsg.hidden = true;
  }
});

Disabled Radio

disabled 속성 추가 시 반투명 처리되고 클릭이 차단됩니다. 그룹 내 일부 항목만 비활성화할 수도 있으며, 비활성 + 선택 상태는 읽기 전용으로 현재 값을 표시할 때 사용합니다.

전체 비활성 그룹
일부 항목 비활성

준비 중인 항목은 선택할 수 없습니다. 

<!-- 전체 비활성: 모든 라디오에 disabled -->
<fieldset style="border: none; padding: 0;">
  <legend class="radio-group-label">전체 비활성 그룹</legend>
  <div class="radio-group">
    <div>
      <input type="radio" data-radio
        id="dis-1" name="dis" value="opt1" disabled />
      <label data-radio-label for="dis-1">옵션 1</label>
    </div>
    <div>
      <input type="radio" data-radio
        id="dis-2" name="dis" value="opt2" disabled checked />
      <label data-radio-label for="dis-2">옵션 2 (선택됨)</label>
    </div>
  </div>
</fieldset>

<!-- 일부 비활성: 특정 항목만 disabled -->
<fieldset style="border: none; padding: 0;">
  <legend class="radio-group-label">일부 항목 비활성</legend>
  <div class="radio-group" aria-describedby="partial-helper">
    <div>
      <input type="radio" data-radio
        id="plan-basic" name="plan" value="basic" checked />
      <label data-radio-label for="plan-basic">기본 플랜</label>
    </div>
    <div>
      <input type="radio" data-radio
        id="plan-ent" name="plan" value="enterprise" disabled />
      <label data-radio-label for="plan-ent">Enterprise (준비 중)</label>
    </div>
  </div>
  <p class="radio-helper" id="partial-helper">
    준비 중인 항목은 선택할 수 없습니다.
  </p>
</fieldset>
/* 비활성 상태: 반투명 + 커서 변경 + 클릭 차단 */
input[type="radio"][data-radio]:disabled + label[data-radio-label] {
  opacity: 0.5;
  cursor: not-allowed;
  pointer-events: none;
}

/* 비활성 + 선택 상태: primary 색상 유지 */
input[type="radio"][data-radio]:disabled:checked
  + label[data-radio-label]::before {
  border-color: var(--color-primary);
}
input[type="radio"][data-radio]:disabled:checked
  + label[data-radio-label]::after {
  opacity: 1;
}
// 조건부 비활성화 예시
const enterpriseRadio = document.querySelector('#plan-ent');
const launchDate = new Date('2026-06-01');
const today = new Date();

// 출시일 이전에는 자동으로 비활성화
if (today < launchDate) {
  enterpriseRadio.disabled = true;
}

Radio Group Layout

.radio-group은 수직(세로) 배치, .radio-group--inline은 수평(가로) 배치입니다. 옵션이 3개 이하이고 텍스트가 짧을 때 수평 레이아웃이 효과적입니다.

수직 레이아웃 (기본)
수평 레이아웃 (inline)
만족도 평가

서비스 이용 후 만족도를 선택해 주세요. 

<!-- 수직 레이아웃: .radio-group (기본) -->
<fieldset style="border: none; padding: 0;">
  <legend class="radio-group-label">사이즈 선택</legend>
  <div class="radio-group">
    <div>
      <input type="radio" data-radio id="sz-s" name="size" value="S" checked />
      <label data-radio-label for="sz-s">소형 (S)</label>
    </div>
    <div>
      <input type="radio" data-radio id="sz-m" name="size" value="M" />
      <label data-radio-label for="sz-m">중형 (M)</label>
    </div>
    <div>
      <input type="radio" data-radio id="sz-l" name="size" value="L" />
      <label data-radio-label for="sz-l">대형 (L)</label>
    </div>
  </div>
</fieldset>

<!-- 수평 레이아웃: .radio-group.radio-group--inline -->
<fieldset style="border: none; padding: 0;">
  <legend class="radio-group-label">성별</legend>
  <div class="radio-group radio-group--inline">
    <div>
      <input type="radio" data-radio id="g-m" name="gender" value="male" checked />
      <label data-radio-label for="g-m">남성</label>
    </div>
    <div>
      <input type="radio" data-radio id="g-f" name="gender" value="female" />
      <label data-radio-label for="g-f">여성</label>
    </div>
    <div>
      <input type="radio" data-radio id="g-o" name="gender" value="other" />
      <label data-radio-label for="g-o">기타</label>
    </div>
  </div>
</fieldset>
/* 수직 그룹 (기본) */
.radio-group {
  display: flex;
  flex-direction: column;
  gap: var(--spacing-sm);
}

/* 수평 그룹: .radio-group에 추가 */
.radio-group--inline {
  flex-direction: row;
  flex-wrap: wrap;
  gap: var(--spacing-md);
  align-items: center;
}

/* 모바일: 수평도 수직으로 전환 */
@media (max-width: 767px) {
  .radio-group--inline {
    flex-direction: column;
    gap: var(--spacing-sm);
  }
}
// 레이아웃 동적 전환 예시
const radioGroup = document.querySelector('.radio-group');
const toggleBtn  = document.querySelector('#layout-toggle');

toggleBtn.addEventListener('click', () => {
  radioGroup.classList.toggle('radio-group--inline');
  toggleBtn.textContent =
    radioGroup.classList.contains('radio-group--inline')
      ? '수직으로 전환'
      : '수평으로 전환';
});
