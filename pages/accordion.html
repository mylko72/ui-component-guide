Accordion

콘텐츠를 접었다 펼칠 수 있는 컴포넌트입니다. WAI-ARIA Accordion 패턴을 준수하며 data-component="accordion"으로 컨테이너를 지정하고, data-accordion-trigger 버튼과 data-accordion-panel 패널을 aria-controls / aria-expanded로 연결합니다. JavaScript 없이도 모든 패널이 펼쳐진 상태로 접근 가능합니다.

Default Accordion

기본 아코디언입니다. 여러 패널을 동시에 열 수 있는 다중 열림 모드(multiple)가 기본값입니다. JavaScript 없이도 모든 패널이 펼쳐진 상태로 접근할 수 있습니다.

국내 배송은 주문 후 2~3 영업일 이내 발송되며, 제주 및 도서 산간 지역은 추가 1~2일이 소요될 수 있습니다. 3만 원 이상 구매 시 무료 배송이 적용됩니다.

상품 수령 후 7일 이내 반품 및 교환이 가능합니다. 단, 사용 흔적이 있거나 포장이 훼손된 경우에는 불가합니다. 고객센터(1234-5678)로 연락 주시면 안내해 드리겠습니다.

신용카드, 체크카드, 무통장 입금, 카카오페이, 네이버페이, 토스페이 등 다양한 결제 수단을 지원합니다. 할부는 신용카드 3~12개월 무이자 할부가 가능합니다. 
<!-- data-component="accordion": Accordion.init() 초기화 대상 -->
<div class="accordion" data-component="accordion"
  data-accordion-mode="multiple">

  <div class="accordion__item">
    <h3 class="accordion__heading">
      <button
        class="accordion__trigger"
        type="button"
        data-accordion-trigger
        aria-expanded="true"
        aria-controls="panel-1"
        id="trigger-1"
      >
        배송 정책은 어떻게 되나요?
        <span class="accordion__icon" aria-hidden="true">
          <i data-lucide="chevron-down"></i>
        </span>
      </button>
    </h3>
    <div
      id="panel-1"
      class="accordion__panel"
      data-accordion-panel
      role="region"
      aria-labelledby="trigger-1"
    >
      <div class="accordion__content">
        패널 내용...
      </div>
    </div>
  </div>

</div>
/* aria-expanded 상태로 아이콘 제어 */
:where([aria-expanded='true']) .accordion__icon {
  transform: rotate(180deg);
  color: var(--color-primary);
}

:where([aria-expanded='true']).accordion__trigger {
  color: var(--color-primary);
}

/* 트리거 호버 */
.accordion__trigger:hover {
  background-color: var(--color-bg-secondary);
}

/* 포커스 가시성 */
.accordion__trigger:focus-visible {
  outline: 2px solid var(--color-primary);
  outline-offset: -2px;
}
// DOMContentLoaded 시 자동 초기화
document.addEventListener('DOMContentLoaded', () => {
  Accordion.init(); // 페이지의 모든 [data-component="accordion"] 초기화
});

// 특정 컨테이너 범위만 초기화:
Accordion.init(document.querySelector('.my-container'));

Single Mode

data-accordion-mode="single"을 지정하면 한 번에 하나의 패널만 열 수 있습니다. 다른 항목을 클릭하면 열린 항목이 자동으로 닫힙니다.

홈페이지 우측 상단 '회원가입' 버튼을 클릭하거나 소셜 로그인(카카오, 네이버, Google)으로 간편하게 가입할 수 있습니다. 이메일 인증 후 즉시 서비스를 이용하실 수 있습니다.

로그인 페이지의 '비밀번호 찾기' 링크를 클릭하여 가입한 이메일 주소를 입력하면 재설정 링크를 받을 수 있습니다. 링크의 유효 시간은 30분입니다.

탈퇴 후 30일 이내에는 데이터 복구가 가능하며, 30일 경과 후에는 모든 개인정보가 완전히 삭제됩니다. 다만, 법적 보관 의무가 있는 거래 내역은 관련 법령에 따라 보관됩니다. 
<!-- data-accordion-mode="single": 단일 열림 모드 -->
<div class="accordion" data-component="accordion"
  data-accordion-mode="single">

  <div class="accordion__item">
    <h3 class="accordion__heading">
      <button
        class="accordion__trigger"
        type="button"
        data-accordion-trigger
        aria-expanded="true"
        aria-controls="panel-1"
        id="trigger-1"
      >
        회원가입은 어떻게 하나요?
        <span class="accordion__icon" aria-hidden="true">
          <i data-lucide="chevron-down"></i>
        </span>
      </button>
    </h3>
    <div id="panel-1" class="accordion__panel"
      data-accordion-panel
      role="region"
      aria-labelledby="trigger-1">
      <div class="accordion__content">내용...</div>
    </div>
  </div>

</div>
/* 단일 모드는 JS가 동작 제어, 별도 CSS 없음 */
/* aria-expanded 상태로 아이콘 제어 */
:where([aria-expanded='true']) .accordion__icon {
  transform: rotate(180deg);
  color: var(--color-primary);
}
// data-accordion-mode="single" 속성만으로 단일 모드 활성화
// Accordion.init()이 자동으로 감지하여 처리

// Single 모드 동작:
// - 열린 항목 클릭 → 닫힘
// - 닫힌 항목 클릭 → 나머지 모두 닫히고 해당 항목만 열림

Flush Accordion

외부 테두리와 둥근 모서리를 제거한 Flush 변형입니다. .accordion--flush 클래스를 추가하면 적용됩니다. 페이지 콘텐츠 영역이나 카드 내부에 삽입할 때 적합합니다.

HyperText Markup Language의 약자로 웹 페이지의 구조를 정의하는 마크업 언어입니다. 시맨틱 태그를 활용하면 검색 엔진 최적화(SEO)와 접근성을 향상시킬 수 있습니다.

Cascading Style Sheets는 HTML 요소의 표시 방식을 지정합니다. 선택자, 속성, 값의 조합으로 스타일 규칙을 작성하며, CSS 변수(Custom Properties)를 사용하면 일관된 디자인 시스템을 구축할 수 있습니다.

JavaScript는 웹 페이지에 동적인 기능을 추가하는 프로그래밍 언어입니다. 사용자 이벤트 처리, DOM 조작, 비동기 데이터 처리(AJAX) 등을 담당합니다. 점진적 강화(Progressive Enhancement) 원칙에 따라 JS 없이도 기본 기능이 동작해야 합니다. 
<!-- .accordion--flush: 외부 테두리 없는 변형 -->
<div class="accordion accordion--flush"
  data-component="accordion"
  data-accordion-mode="multiple"
>
  <div class="accordion__item">
    <h3 class="accordion__heading">
      <button
        class="accordion__trigger"
        type="button"
        data-accordion-trigger
        aria-expanded="true"
        aria-controls="panel-flush-1"
        id="trigger-flush-1"
      >
        HTML이란 무엇인가요?
        <span class="accordion__icon" aria-hidden="true">
          <i data-lucide="chevron-down"></i>
        </span>
      </button>
    </h3>
    <div id="panel-flush-1" class="accordion__panel"
      data-accordion-panel
      role="region"
      aria-labelledby="trigger-flush-1"
    >
      <div class="accordion__content">내용...</div>
    </div>
  </div>
</div>
/* Flush 변형 */
.accordion--flush {
  border: none;
  border-radius: 0;
}

.accordion--flush .accordion__item {
  border-bottom: 1px solid var(--color-border);
}

.accordion--flush .accordion__item:last-child {
  border-bottom: none;
}
// Flush 아코디언도 동일하게 Accordion.init()으로 자동 초기화
// .accordion--flush는 CSS 스타일 변형만 담당, JS 동작은 동일

Disabled Item

개별 항목을 비활성화하려면 트리거 버튼에 disabled 속성을 사용합니다. 비활성 항목은 클릭과 키보드 이동에서 모두 건너뜁니다. aria-disabled="true"를 함께 사용해 스크린 리더에도 상태를 전달합니다.

본 서비스는 만 14세 이상 사용자에게 제공됩니다. 서비스 이용 시 관련 법령 및 본 약관을 준수해야 합니다. 위반 시 서비스 이용이 제한될 수 있습니다.

개인정보 처리방침 내용이 준비 중입니다.

본 사이트는 서비스 개선을 위해 쿠키를 사용합니다. 필수 쿠키는 서비스 운영에 반드시 필요하며, 선택적 쿠키는 사용자 설정에서 비활성화할 수 있습니다. 
<!-- 비활성 항목: disabled + aria-disabled="true" -->
<div class="accordion" data-component="accordion">

  <!-- 정상 항목 -->
  <div class="accordion__item">
    <h3 class="accordion__heading">
      <button
        class="accordion__trigger"
        type="button"
        data-accordion-trigger
        aria-expanded="true"
        aria-controls="panel-1"
        id="trigger-1"
      >이용 약관</button>
    </h3>
    <div id="panel-1" class="accordion__panel"
      data-accordion-panel
      role="region"
      aria-labelledby="trigger-1">
      <div class="accordion__content">내용...</div>
    </div>
  </div>

  <!--
    비활성 항목:
    - disabled: 클릭 차단 (네이티브)
    - aria-disabled="true": 스크린 리더에 상태 전달
    - accordion.js가 Arrow 키 이동 시 자동으로 건너뜀
  -->
  <div class="accordion__item">
    <h3 class="accordion__heading">
      <button
        class="accordion__trigger"
        type="button"
        data-accordion-trigger
        aria-expanded="true"
        aria-controls="panel-2"
        id="trigger-2"
        disabled
        aria-disabled="true"
      >개인정보 처리방침 (준비 중)</button>
    </h3>
    <div id="panel-2" class="accordion__panel"
      data-accordion-panel
      role="region"
      aria-labelledby="trigger-2">
      <div class="accordion__content">내용...</div>
    </div>
  </div>

</div>
/* 비활성 트리거: 반투명 + 클릭 차단 */
.accordion__trigger[disabled],
.accordion__trigger[aria-disabled="true"] {
  opacity: 0.4;
  cursor: not-allowed;
  pointer-events: none;
}
// 동적으로 항목 비활성화
const trigger = document.getElementById('trigger-2');

// 비활성화
trigger.disabled = true;
trigger.setAttribute('aria-disabled', 'true');

// 활성화
trigger.disabled = false;
trigger.removeAttribute('aria-disabled');
