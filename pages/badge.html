Badge

상태, 카테고리, 레이블을 짧은 텍스트로 표시하는 인라인 요소입니다. data-color로 6가지 색상, data-outlined로 테두리형, data-dot으로 점 인디케이터형, data-size로 sm / md / lg 크기를 지정합니다.

Colors

6가지 의미론적 색상을 제공합니다. data-color 속성으로 색상을 지정합니다. 배경색과 텍스트 색상은 WCAG 2.2 AA 대비비(4.5:1)를 준수합니다.

Neutral Primary Success Warning Error Info 
<span class="badge" data-color="neutral">Neutral</span>
<span class="badge" data-color="primary">Primary</span>
<span class="badge" data-color="success">Success</span>
<span class="badge" data-color="warning">Warning</span>
<span class="badge" data-color="error">Error</span>
<span class="badge" data-color="info">Info</span>
/* Badge 기본: 인라인 플렉스, pill 형태 */
.badge {
  display: inline-flex;
  align-items: center;
  gap: 0.375rem;
  padding: 0.25rem 0.625rem;
  border-radius: 9999px;
  font-size: var(--font-size-xs);
  font-weight: 600;
  line-height: 1;
  white-space: nowrap;
  border: 1px solid transparent;
}

/* neutral (회색) */
.badge[data-color="neutral"] { background-color: #6b7280; color: #ffffff; }

/* primary (파란색) */
.badge[data-color="primary"] { background-color: var(--color-primary); color: #ffffff; }

/* success (초록색) */
.badge[data-color="success"] { background-color: var(--color-success); color: #ffffff; }

/* warning (노란색) — 어두운 텍스트로 대비 확보 */
.badge[data-color="warning"] { background-color: var(--color-warning); color: #1f2937; }

/* error (빨간색) */
.badge[data-color="error"] { background-color: var(--color-danger); color: #ffffff; }

/* info (밝은 파란색) */
.badge[data-color="info"] { background-color: var(--color-info); color: #ffffff; }
/* Badge는 순수 CSS 컴포넌트로 JS가 필요하지 않습니다.
   data-color 속성을 동적으로 변경하여 상태를 표현할 수 있습니다. */
const badge = document.querySelector('.badge');
badge.dataset.color = 'success'; /* 동적 색상 변경 */

Outlined

배경이 투명하고 테두리와 텍스트에만 색상을 표현하는 변형입니다. data-outlined="true"를 추가하면 적용됩니다. 라이트한 느낌이 필요할 때 사용합니다.

Neutral Primary Success Warning Error Info 
<span class="badge" data-color="neutral" data-outlined="true">Neutral</span>
<span class="badge" data-color="primary" data-outlined="true">Primary</span>
<span class="badge" data-color="success" data-outlined="true">Success</span>
<span class="badge" data-color="warning" data-outlined="true">Warning</span>
<span class="badge" data-color="error" data-outlined="true">Error</span>
<span class="badge" data-color="info" data-outlined="true">Info</span>
/* Outlined 변형: 배경 투명, 테두리+텍스트 색상 */
.badge[data-outlined="true"] {
  background-color: transparent;
}

/* neutral outlined */
.badge[data-color="neutral"][data-outlined="true"] {
  border-color: #6b7280;
  color: #6b7280;
}

/* primary outlined */
.badge[data-color="primary"][data-outlined="true"] {
  border-color: var(--color-primary);
  color: var(--color-primary);
}

/* success outlined */
.badge[data-color="success"][data-outlined="true"] {
  border-color: var(--color-success);
  color: var(--color-success);
}

/* warning outlined — 명도 낮춰 대비비 확보 */
.badge[data-color="warning"][data-outlined="true"] {
  border-color: var(--color-warning);
  color: #b45309;
}

/* error outlined */
.badge[data-color="error"][data-outlined="true"] {
  border-color: var(--color-danger);
  color: var(--color-danger);
}

/* info outlined */
.badge[data-color="info"][data-outlined="true"] {
  border-color: var(--color-info);
  color: var(--color-info);
}
/* Outlined Badge는 CSS 전용입니다.
   data-outlined 속성을 토글하여 Filled/Outlined 전환 가능 */
const badge = document.querySelector('.badge');
const isOutlined = badge.dataset.outlined === 'true';
badge.dataset.outlined = isOutlined ? 'false' : 'true';

Dot

텍스트 앞에 작은 원형 점 인디케이터를 표시합니다. data-dot="true"를 추가하면 ::before 가상 요소로 점이 렌더링됩니다. 온라인 상태, 활성/비활성, 버전 표시 등에 활용합니다.

Neutral Primary Active Warning Offline Info 
<span class="badge" data-color="neutral" data-dot="true">Neutral</span>
<span class="badge" data-color="primary" data-dot="true">Primary</span>
<span class="badge" data-color="success" data-dot="true">Active</span>
<span class="badge" data-color="warning" data-dot="true">Warning</span>
<span class="badge" data-color="error" data-dot="true">Offline</span>
<span class="badge" data-color="info" data-dot="true">Info</span>
/* Dot 변형: ::before 가상 요소로 원형 점 */
.badge[data-dot="true"]::before {
  content: "";
  display: inline-block;
  width: 0.5rem;
  height: 0.5rem;
  border-radius: 50%;
  flex-shrink: 0;
  background-color: currentColor;
}

/* Filled Badge의 dot은 반투명 흰색 */
.badge[data-color]:not([data-outlined="true"])[data-dot="true"]::before {
  background-color: rgba(255, 255, 255, 0.8);
}

/* warning Filled dot */
.badge[data-color="warning"]:not([data-outlined="true"])[data-dot="true"]::before {
  background-color: rgba(31, 41, 55, 0.6);
}
/* Dot Badge는 CSS 전용입니다.
   data-dot 속성으로 점 표시 여부를 제어합니다. */
const badge = document.querySelector('.badge');
badge.dataset.dot = 'true'; /* 점 표시 활성화 */

Sizes

3가지 크기 변형을 제공합니다. data-size="sm" / data-size="md" / data-size="lg"로 지정합니다. 기본 크기는 md입니다.

Small (sm)

Neutral Primary Success Warning Error Info

Medium (md)

Neutral Primary Success Warning Error Info

Large (lg)

Neutral Primary Success Warning Error Info 
<!-- Small -->
<span class="badge" data-color="primary" data-size="sm">Small</span>

<!-- Medium (기본값) -->
<span class="badge" data-color="primary" data-size="md">Medium</span>
<span class="badge" data-color="primary">Medium (기본)</span>

<!-- Large -->
<span class="badge" data-color="primary" data-size="lg">Large</span>
/* 크기 변형: sm */
.badge[data-size="sm"] {
  padding: 0.125rem 0.5rem;
  font-size: 0.6875rem; /* 11px */
}

/* 크기 변형: md */
.badge[data-size="md"] {
  padding: 0.25rem 0.625rem;
  font-size: var(--font-size-xs); /* 12px */
}

/* 크기 변형: lg */
.badge[data-size="lg"] {
  padding: 0.375rem 0.75rem;
  font-size: var(--font-size-sm); /* 14px */
}
/* 크기는 CSS data-size 속성으로만 제어합니다. */
const badge = document.querySelector('.badge');

/* sm → md → lg 순환 변경 예시 */
const sizes = ['sm', 'md', 'lg'];
let currentIndex = 0;
badge.dataset.size = sizes[++currentIndex % sizes.length];
