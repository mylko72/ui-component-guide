Tabs

콘텐츠를 여러 패널로 나눠 탭으로 전환하는 컴포넌트입니다. WAI-ARIA Tabs 패턴을 준수하며 role="tablist"를 가진 컨테이너에 탭 버튼(role="tab")과 패널(role="tabpanel")을 연결합니다. 키보드 화살표 키로 탭 간 이동을 지원하고, data-initialized로 중복 초기화를 방지합니다.

Default Tabs

기본 Underline 스타일 탭입니다. role="tablist" 컨테이너에 role="tab" 버튼과 role="tabpanel" 패널을 aria-controls / aria-labelledby로 연결합니다. JavaScript 없이도 첫 번째 탭 내용이 표시됩니다.

프로필 정보

이름, 이메일, 프로필 사진 등 기본 계정 정보를 관리합니다. 변경 사항은 즉시 저장됩니다.

환경 설정

알림, 언어, 시간대 등 개인 설정을 조정합니다. 변경 사항은 다음 로그인 시 적용됩니다.

보안 설정

비밀번호 변경, 2단계 인증, 로그인 기기 관리 등 계정 보안을 강화합니다. 

<!-- tablist: role="tablist", aria-label로 그룹 설명 -->
<div role="tablist" class="tabs" aria-label="계정 정보 탭">
  <!-- tab: aria-selected, aria-controls로 패널 연결 -->
  <button
    role="tab"
    id="tab-profile"
    aria-selected="true"
    aria-controls="panel-profile"
    tabindex="0"
  >프로필</button>

  <button
    role="tab"
    id="tab-settings"
    aria-selected="false"
    aria-controls="panel-settings"
    tabindex="-1"
  >설정</button>
</div>

<!-- tabpanel: aria-labelledby로 탭 버튼 참조 -->
<div id="panel-profile" role="tabpanel"
  aria-labelledby="tab-profile" tabindex="0">
  프로필 내용...
</div>
<div id="panel-settings" role="tabpanel"
  aria-labelledby="tab-settings" tabindex="0"
  style="display:none;">
  설정 내용...
</div>
/* 탭 목록 컨테이너 */
.tabs {
  display: flex;
  border-bottom: 2px solid var(--color-border);
}

/* 탭 버튼: 비활성 */
.tabs [role="tab"] {
  padding: 10px 16px;
  font-size: var(--font-size-sm);
  font-weight: 500;
  color: var(--color-text-secondary);
  background: transparent;
  border: none;
  border-bottom: 2px solid transparent;
  margin-bottom: -2px; /* tablist 하단 border 위에 겹침 */
  cursor: pointer;
  transition:
    color var(--transition-fast),
    border-color var(--transition-fast);
}

/* 탭 버튼: 활성 */
.tabs [role="tab"][aria-selected="true"] {
  color: var(--color-primary);
  border-bottom-color: var(--color-primary);
}

/* 포커스 가시성 */
.tabs [role="tab"]:focus-visible {
  outline: 2px solid var(--color-primary);
  outline-offset: -2px;
}

/* 탭 패널 */
[role="tabpanel"] {
  padding: var(--spacing-lg) 0;
  outline: none;
}
// tabs.js 자동 초기화: DOMContentLoaded 시 Tabs.init() 호출
// 특정 컨테이너 범위만 초기화:
Tabs.init(document.querySelector('.my-container'));

// 탭 전환 이벤트 감지
const tablist = document.querySelector('[role="tablist"]');
tablist.addEventListener('click', (e) => {
  const tab = e.target.closest('[role="tab"]');
  if (tab) {
    console.log('선택된 탭:', tab.textContent.trim());
    console.log('패널 ID:', tab.getAttribute('aria-controls'));
  }
});

Filled Tabs

활성 탭을 배경색으로 강조하는 Filled 변형입니다. .tabs--filled 클래스를 추가하면 활성 탭에 흰색 배경과 그림자가 적용됩니다.

전체 주문 내역을 표시합니다. 최근 30일 기준으로 총 12건의 주문이 있습니다.

현재 처리 중인 주문 3건을 표시합니다. 평균 처리 시간은 2~3일입니다.

완료된 주문 8건을 표시합니다. 배송이 완료된 주문들입니다.

취소된 주문 1건을 표시합니다. 환불 처리가 완료되었습니다. 

<!-- Filled 변형: .tabs--filled 클래스 추가 -->
<div role="tablist" class="tabs tabs--filled" aria-label="주문 상태 탭">
  <button role="tab" id="tab-all"
    aria-selected="true"
    aria-controls="panel-all"
    tabindex="0">전체</button>
  <button role="tab" id="tab-pending"
    aria-selected="false"
    aria-controls="panel-pending"
    tabindex="-1">처리 중</button>
  <button role="tab" id="tab-complete"
    aria-selected="false"
    aria-controls="panel-complete"
    tabindex="-1">완료</button>
</div>

<div id="panel-all" role="tabpanel"
  aria-labelledby="tab-all" tabindex="0">
  전체 주문 내역...
</div>
<div id="panel-pending" role="tabpanel"
  aria-labelledby="tab-pending" tabindex="0"
  style="display:none;">
  처리 중 주문...
</div>
/* Filled 변형: 배경색 컨테이너 */
.tabs--filled {
  background-color: var(--color-bg-secondary);
  border-radius: var(--radius-md) var(--radius-md) 0 0;
  border-bottom: none;
  padding: 4px;
  gap: 2px;
}

/* 탭 버튼 */
.tabs--filled [role="tab"] {
  border-bottom: none;
  margin-bottom: 0;
  border-radius: var(--radius-sm);
}

/* 활성 탭: 흰색 배경 + 그림자 */
.tabs--filled [role="tab"][aria-selected="true"] {
  color: var(--color-text);
  background-color: var(--color-bg);
  box-shadow: var(--shadow-sm);
  border-bottom: none;
}
// Filled 탭도 동일하게 Tabs.init()으로 자동 초기화
// DOMContentLoaded 시 자동으로 실행됨
document.addEventListener('DOMContentLoaded', () => {
  Tabs.init(); // 페이지의 모든 [role="tablist"] 초기화
});

Vertical Tabs

탭을 세로 방향으로 배치하는 Vertical 변형입니다. .tabs--vertical 클래스와 .tabs-vertical-layout 래퍼로 구성합니다. 사이드 네비게이션 패턴에 적합합니다.

시작하기

이 라이브러리는 접근성과 사용성을 중시하는 UI 컴포넌트 모음입니다. 간단한 HTML 마크업만으로 시작할 수 있습니다.

설치 방법

CSS와 JS 파일을 HTML에 포함하거나 CDN을 통해 로드합니다. 별도의 빌드 도구가 필요하지 않습니다.

사용 방법

role="tablist", role="tab", role="tabpanel" ARIA 속성으로 마크업을 구성하면 자동으로 초기화됩니다.

API 레퍼런스

Tabs.init(container)로 특정 영역만 초기화하거나 전체 문서를 대상으로 초기화합니다. 

<!-- 수직 탭: .tabs-vertical-layout 래퍼 안에 구성 -->
<div class="tabs-vertical-layout">

  <!-- .tabs--vertical: 세로 방향 탭 목록 -->
  <div role="tablist" class="tabs tabs--vertical"
    aria-label="문서 카테고리"
    aria-orientation="vertical"
  >
    <button role="tab" id="tab-intro"
      aria-selected="true"
      aria-controls="panel-intro"
      tabindex="0">시작하기</button>
    <button role="tab" id="tab-install"
      aria-selected="false"
      aria-controls="panel-install"
      tabindex="-1">설치</button>
  </div>

  <div id="panel-intro" role="tabpanel"
    aria-labelledby="tab-intro" tabindex="0">
    시작하기 내용...
  </div>
  <div id="panel-install" role="tabpanel"
    aria-labelledby="tab-install" tabindex="0"
    style="display:none;">
    설치 방법...
  </div>

</div>
/* 수직 탭 목록 */
.tabs--vertical {
  flex-direction: column;
  align-items: stretch;
  border-bottom: none;
  border-right: 2px solid var(--color-border);
  min-width: 160px;
}

/* 수직 탭 버튼 */
.tabs--vertical [role="tab"] {
  text-align: left;
  border-bottom: none;
  border-right: 2px solid transparent;
  margin-bottom: 0;
  margin-right: -2px;
  border-radius: var(--radius-sm) 0 0 var(--radius-sm);
}

/* 활성 수직 탭 */
.tabs--vertical [role="tab"][aria-selected="true"] {
  border-right-color: var(--color-primary);
  border-bottom-color: transparent;
}

/* 탭 + 패널 가로 배치 */
.tabs-vertical-layout {
  display: flex;
  align-items: flex-start;
}

.tabs-vertical-layout [role="tabpanel"] {
  flex: 1;
  padding: 0 var(--spacing-lg);
}
// 수직 탭도 Tabs.init()으로 자동 초기화
// aria-orientation="vertical"은 스크린 리더에게
// 탭이 수직 방향임을 알려주는 ARIA 속성입니다
// (현재 js에서 Arrow Up/Down 방향 키를 별도 처리하지 않음)

Disabled Tab

개별 탭을 비활성화하려면 disabled 속성을 사용합니다. 비활성 탭은 클릭과 키보드 이동에서 모두 건너뜁니다. aria-disabled="true"를 함께 사용해 스크린 리더에도 상태를 전달합니다.

서비스의 전체 현황을 한눈에 확인합니다. 주요 지표와 최근 활동 내역을 표시합니다.

사용자 행동 분석, 방문자 통계 등 상세 분석 데이터를 확인합니다.

리포트 기능은 준비 중입니다.

데이터를 CSV, Excel 등 다양한 형식으로 내보냅니다. 

<!-- 비활성 탭: disabled + aria-disabled="true" -->
<div role="tablist" class="tabs" aria-label="서비스 탭">
  <button role="tab" id="tab-overview"
    aria-selected="true"
    aria-controls="panel-overview"
    tabindex="0">개요</button>

  <button role="tab" id="tab-analytics"
    aria-selected="false"
    aria-controls="panel-analytics"
    tabindex="-1">분석</button>

  <!--
    비활성 탭:
    - disabled: 클릭 차단 (네이티브)
    - aria-disabled="true": 스크린 리더에 상태 전달
    - tabs.js가 Arrow 키 이동 시 자동으로 건너뜀
  -->
  <button role="tab" id="tab-reports"
    aria-selected="false"
    aria-controls="panel-reports"
    tabindex="-1"
    disabled
    aria-disabled="true">리포트 (준비 중)</button>
</div>
/* 비활성 탭: 반투명 + 클릭 차단 */
.tabs [role="tab"][disabled],
.tabs [role="tab"][aria-disabled="true"] {
  opacity: 0.4;
  cursor: not-allowed;
  pointer-events: none;
}
// 동적으로 탭 비활성화
const reportTab = document.getElementById('tab-reports');

// 비활성화
reportTab.disabled = true;
reportTab.setAttribute('aria-disabled', 'true');
reportTab.setAttribute('tabindex', '-1');

// 활성화
reportTab.disabled = false;
reportTab.removeAttribute('aria-disabled');
reportTab.setAttribute('tabindex', '0');

Tabs with Icon

탭 버튼에 아이콘을 포함하여 시각적 식별성을 높입니다. 아이콘은 장식적 요소이므로 aria-hidden="true"를 적용하고, 탭 레이블 텍스트로 의미를 전달합니다.

홈 대시보드 화면입니다. 전체 현황을 한눈에 확인할 수 있습니다.

사용자 관리 화면입니다. 계정 목록과 권한을 설정합니다.

알림 설정 화면입니다. 이메일, 푸시 알림 수신 여부를 조정합니다.

시스템 설정 화면입니다. 언어, 시간대, 테마를 변경합니다. 

<!-- 아이콘 + 텍스트 탭 -->
<div role="tablist" class="tabs" aria-label="대시보드 탭">

  <button role="tab" id="tab-home"
    aria-selected="true"
    aria-controls="panel-home"
    tabindex="0">
    <!-- 장식적 아이콘: aria-hidden으로 스크린 리더 무시 -->
    <i data-lucide="home" aria-hidden="true"></i>
    홈
  </button>

  <button role="tab" id="tab-user"
    aria-selected="false"
    aria-controls="panel-user"
    tabindex="-1">
    <i data-lucide="user" aria-hidden="true"></i>
    사용자
  </button>

  <button role="tab" id="tab-settings"
    aria-selected="false"
    aria-controls="panel-settings"
    tabindex="-1">
    <i data-lucide="settings" aria-hidden="true"></i>
    설정
  </button>

</div>
/* 아이콘 + 텍스트 정렬 */
.tabs [role="tab"] {
  display: inline-flex;
  align-items: center;
  gap: 6px;
}

/* 아이콘 크기 */
.tabs [role="tab"] svg,
.tabs [role="tab"] i[data-lucide] {
  width: 14px;
  height: 14px;
  flex-shrink: 0;
}

/* 아이콘만 있는 탭 (텍스트 없음) */
.tabs [role="tab"].tab--icon-only {
  padding: 10px;
}

/* aria-label로 의미를 제공하는 아이콘 전용 탭 */
.tabs [role="tab"][aria-label] svg {
  width: 16px;
  height: 16px;
}
// 아이콘 탭도 Tabs.init()으로 자동 초기화
// Lucide 아이콘은 common.js의 initIcons()가 렌더링

// 아이콘만 있는 탭을 사용할 경우:
// aria-label 속성으로 의미를 명시적으로 제공해야 합니다
const iconOnlyTabs = document.querySelectorAll('[role="tab"].tab--icon-only');
iconOnlyTabs.forEach((tab, i) => {
  const labels = ['홈', '사용자', '설정'];
  tab.setAttribute('aria-label', labels[i]);
});
