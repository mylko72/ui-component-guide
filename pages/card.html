Card

콘텐츠를 그룹화하고 시각적으로 분리하는 컨테이너 컴포넌트입니다. BEM 패턴의 .card, .card__header, .card__body, .card__footer 구조로 구성되며, .card--interactive로 호버 효과를, .card--horizontal로 가로형 레이아웃을 적용할 수 있습니다. JavaScript 없이 CSS만으로 완전히 동작합니다.

기본 Card

가장 기본적인 카드 형태입니다. .card 컨테이너 안에 .card__body를 배치하고, .card__title과 .card__description으로 콘텐츠를 구성합니다. 색상 변형 클래스(--primary, --success, --warning, --error)로 의미론적 강조를 추가할 수 있습니다.

기본 카드

가장 심플한 카드 형태입니다. 헤더와 푸터 없이 본문만 표시합니다. 콘텐츠를 그룹화하여 시각적으로 분리하는 용도로 사용합니다.

Primary 카드

.card--primary 클래스로 파란색 계열 테두리 강조를 적용합니다. 주요 정보나 행동 유도(CTA)에 활용하세요.

Success 카드

.card--success 클래스로 초록색 계열 강조를 적용합니다. 완료 상태, 성공 메시지 표시에 적합합니다.

Warning 카드

.card--warning 클래스로 노란색 계열 강조를 적용합니다. 주의가 필요한 정보나 경고 메시지에 사용합니다. 

<!-- 기본 카드 구조 -->
<article class="card">
  <div class="card__body">
    <h3 class="card__title">카드 제목</h3>
    <p class="card__description">
      카드 설명 텍스트입니다.
    </p>
  </div>
</article>

<!-- 색상 변형: primary / success / warning / error -->
<article class="card card--primary">
  <div class="card__body">
    <h3 class="card__title">Primary 카드</h3>
    <p class="card__description">파란색 강조 테두리</p>
  </div>
</article>

<!-- 반응형 그리드 배치 -->
<div class="card-grid">
  <article class="card">...</article>
  <article class="card card--success">...</article>
  <article class="card card--warning">...</article>
</div>
/* 베이스 카드 */
.card {
  display: flex;
  flex-direction: column;
  border: 1px solid var(--color-border);
  border-radius: var(--radius-md);
  background-color: var(--color-bg);
  box-shadow: var(--shadow-sm);
  overflow: hidden;
}

/* 본문 영역 */
.card__body {
  padding: var(--spacing-md);
  flex: 1;
  display: flex;
  flex-direction: column;
  gap: var(--spacing-xs);
}

/* 제목 */
.card__title {
  font-size: var(--font-size-lg);  /* 1.125rem / 18px */
  font-weight: 600;
  color: var(--color-text);
}

/* 설명 텍스트 */
.card__description {
  font-size: var(--font-size-sm);
  color: var(--color-text-secondary);
  margin-top: var(--spacing-xs);
}

/* 색상 변형 */
.card--primary {
  border-color: var(--color-primary);
  border-left: 4px solid var(--color-primary);
}
.card--success {
  border-color: var(--color-success);
  border-left: 4px solid var(--color-success);
}
.card--warning {
  border-color: var(--color-warning);
  border-left: 4px solid var(--color-warning);
}
.card--error {
  border-color: var(--color-danger);
  border-left: 4px solid var(--color-danger);
}

/* 반응형 그리드 */
.card-grid {
  display: grid;
  gap: var(--spacing-md);
  grid-template-columns: 1fr;
}
@media (min-width: 640px) {
  .card-grid { grid-template-columns: repeat(2, 1fr); }
}
@media (min-width: 1024px) {
  .card-grid { grid-template-columns: repeat(3, 1fr); }
}

헤더와 푸터를 포함한 Card

.card__header로 상단 강조 영역을, .card__footer로 하단 액션 영역을 추가합니다. 헤더에는 커버 이미지(.card__header--image) 또는 색상 배경(.card__header--primary)을 적용할 수 있습니다. 푸터는 .card__footer--end로 우측 정렬, .card__footer--between으로 양끝 정렬이 가능합니다.

UI 컴포넌트 패턴

재사용 가능한 UI 컴포넌트를 설계하는 패턴과 원칙을 소개합니다.

5분 읽기 
<!-- 헤더 + 바디 + 푸터 전체 구조 -->
<article class="card">

  <!-- 색상 배경 헤더 -->
  <header class="card__header card__header--primary">
    <span>추천 콘텐츠</span>
  </header>

  <!-- 본문 -->
  <div class="card__body">
    <h3 class="card__title">카드 제목</h3>
    <p class="card__description">카드 설명입니다.</p>
  </div>

  <!-- 우측 정렬 푸터 -->
  <footer class="card__footer card__footer--end">
    <button type="button">자세히 보기</button>
  </footer>

</article>

<!-- 이미지 헤더 변형 -->
<article class="card">
  <header class="card__header--image">
    <img src="cover.jpg" alt="커버 이미지" class="card__cover">
  </header>
  <div class="card__body">...</div>
  <footer class="card__footer card__footer--between">
    <span>5분 읽기</span>
    <button type="button">읽기</button>
  </footer>
</article>
/* 헤더 영역 */
.card__header {
  padding: var(--spacing-md);
  border-bottom: 1px solid var(--color-border);
  background-color: var(--color-bg-secondary);
  display: flex;
  align-items: center;
  gap: var(--spacing-sm);
}

/* 색상 배경 헤더 */
.card__header--primary {
  background-color: var(--color-primary);
  color: #ffffff;
  border-bottom-color: transparent;
}

/* 이미지 전용 헤더 */
.card__header--image {
  padding: 0;
  overflow: hidden;
}
.card__header--image img,
.card__header--image .card__cover {
  width: 100%;
  height: 180px;
  object-fit: cover;
  display: block;
}

/* 푸터 영역 */
.card__footer {
  padding: var(--spacing-md);
  border-top: 1px solid var(--color-border);
  background-color: var(--color-bg-secondary);
  display: flex;
  align-items: center;
  gap: var(--spacing-sm);
}

/* 푸터 정렬 변형 */
.card__footer--end     { justify-content: flex-end; }
.card__footer--between { justify-content: space-between; }

가로형 Card — 이미지 + 텍스트

.card--horizontal 클래스로 이미지와 콘텐츠를 좌우로 나란히 배치합니다. .card__image는 좌측 고정 너비(160px)로 표시되며, 모바일(480px 이하)에서는 자동으로 세로형으로 전환됩니다.

컴포넌트 기반 설계

재사용 가능한 UI 컴포넌트를 중심으로 인터페이스를 설계하면 유지보수성과 일관성이 크게 향상됩니다.

CSS BEM

접근성 우선 개발

모든 사용자를 위한 접근 가능한 인터페이스를 처음부터 설계하는 WAI-ARIA 가이드라인 준수 방법을 소개합니다.

WAI-ARIA 2.1

레이어드 아키텍처

토큰 → 기본 스타일 → 컴포넌트 스타일로 이어지는 CSS 계층 구조로 테마 변경과 유지보수가 쉬운 스타일 시스템을 구성합니다. 

<!-- 가로형 카드: .card--horizontal -->
<article class="card card--horizontal">

  <!-- 좌측 이미지 (160px 고정 너비) -->
  <img
    class="card__image"
    src="thumbnail.jpg"
    alt="카드 이미지 설명"
    width="160"
    height="160"
  >

  <!-- 우측 콘텐츠 -->
  <div class="card__body">
    <h3 class="card__title">카드 제목</h3>
    <p class="card__description">카드 설명입니다.</p>
  </div>

</article>

<!-- 푸터 포함 가로형 카드 -->
<article class="card card--horizontal">
  <img class="card__image" src="thumbnail.jpg" alt="이미지">
  <div style="display:flex;flex-direction:column;flex:1;">
    <div class="card__body">
      <h3 class="card__title">카드 제목</h3>
      <p class="card__description">설명</p>
    </div>
    <footer class="card__footer card__footer--between">
      <span>부가 정보</span>
      <button type="button">자세히</button>
    </footer>
  </div>
</article>
/* 가로형 카드 */
.card--horizontal {
  flex-direction: row;
}

/* 좌측 이미지 */
.card--horizontal .card__image {
  width: 160px;
  min-width: 160px;
  max-width: 160px;
  height: auto;
  object-fit: cover;
  border-radius: var(--radius-md) 0 0 var(--radius-md);
  display: block;
}

/* 우측 콘텐츠 */
.card--horizontal .card__body {
  flex: 1;
  min-width: 0;
}

/* 모바일: 세로형 전환 (480px 이하) */
@media (max-width: 480px) {
  .card--horizontal {
    flex-direction: column;
  }
  .card--horizontal .card__image {
    width: 100%;
    min-width: unset;
    height: 180px;
    border-radius: var(--radius-md) var(--radius-md) 0 0;
  }
}

인터랙티브 Card — 호버 효과

.card--interactive 클래스를 추가하면 마우스 호버 시 그림자가 커지고 카드가 살짝 위로 이동합니다. CSS transition과 transform만을 사용하며 JavaScript가 전혀 필요 없습니다. 키보드 탐색 시 :focus-visible로 명확한 포커스 링을 표시합니다. 마우스를 올려보세요.

빠른 성능

최적화된 렌더링 파이프라인으로 60fps의 부드러운 사용자 경험을 제공합니다.

반응형 디자인

모바일, 태블릿, 데스크톱 모든 화면 크기에 자동으로 최적화됩니다.

접근성 보장

WCAG 2.1 AA 기준을 준수하며, 키보드 탐색과 스크린 리더를 완전히 지원합니다.

Tab 키로 포커스 이동 후 Enter/Space로 활성화 

<!-- 인터랙티브 카드: .card--interactive -->
<!-- tabindex="0": 키보드 포커스 가능 -->
<article
  class="card card--interactive"
  tabindex="0"
  aria-label="클릭 가능한 카드"
>
  <div class="card__body">
    <h3 class="card__title">인터랙티브 카드</h3>
    <p class="card__description">
      마우스를 올리면 효과가 나타납니다.
    </p>
  </div>
</article>

<!-- 링크 카드: tabindex 불필요 -->
<a
  href="/detail"
  class="card card--interactive"
  aria-label="상세 페이지로 이동"
>
  <div class="card__body">
    <h3 class="card__title">링크 카드</h3>
    <p class="card__description">클릭하면 이동합니다.</p>
  </div>
</a>
/* 인터랙티브 카드 */
.card--interactive {
  cursor: pointer;
  /* transition은 .card 기본 스타일에서 정의:
     box-shadow 200ms ease, transform 200ms ease */
}

/* 호버: 그림자 증가 + 위로 이동 */
.card--interactive:hover {
  box-shadow: var(--shadow-lg);
  transform: translateY(-3px);
  border-color: var(--color-border-hover);
}

/* 눌림: 살짝 내려옴 */
.card--interactive:active {
  transform: translateY(-1px);
  box-shadow: var(--shadow-md);
}

/* 키보드 포커스 링 */
.card--interactive:focus-visible {
  outline: 2px solid var(--color-primary);
  outline-offset: 2px;
}

/* 모션 감소 설정 준수 */
@media (prefers-reduced-motion: reduce) {
  .card--interactive {
    transition: none;
  }
  .card--interactive:hover {
    transform: none;
  }
}
