Progress

작업 진행 상태를 시각적으로 표현하는 컴포넌트입니다. WAI-ARIA progressbar 패턴을 준수하며 data-component="progress"로 컨테이너를 지정합니다. role="progressbar", aria-valuenow, aria-valuemin, aria-valuemax로 스크린 리더에 진행률을 전달합니다. JavaScript 없이도 CSS 변수 기반으로 진행률이 시각적으로 표시됩니다.

Default Progress

기본 Progress 바입니다. 라벨과 퍼센트 텍스트를 함께 표시합니다. JavaScript 없이도 style="--progress-value:N"으로 너비가 표시됩니다.

초기화 0%
파일 다운로드 25%
데이터 처리 50%
업로드 75%
설치 완료 100% 
<div class="progress-wrapper">
  <div class="progress-header">
    <span class="progress-label" id="progress-label-1">파일 다운로드</span>
    <span class="progress-value-text" aria-hidden="true">75%</span>
  </div>
  <div
    class="progress"
    data-component="progress"
    data-progress="75"
    role="progressbar"
    aria-valuenow="75"
    aria-valuemin="0"
    aria-valuemax="100"
    aria-labelledby="progress-label-1"
  >
    <div class="progress__bar" style="--progress-value: 75;"></div>
  </div>
</div>
/* 트랙: 회색 배경 컨테이너 */
.progress {
  position: relative;
  width: 100%;
  height: 0.5rem; /* 8px */
  background-color: var(--color-border);
  border-radius: 9999px;
  overflow: hidden;
}

/* 바: CSS 변수(--progress-value)로 너비 제어 */
.progress__bar {
  height: 100%;
  border-radius: inherit;
  background-color: var(--color-primary);
  width: calc(var(--progress-value, 0) * 1%);
  transition: width var(--transition-normal);
}

/* 래퍼 레이아웃 */
.progress-wrapper {
  display: flex;
  flex-direction: column;
  gap: 0.375rem;
  width: 100%;
}

.progress-header {
  display: flex;
  align-items: center;
  justify-content: space-between;
}
// DOMContentLoaded 시 자동 초기화
document.addEventListener('DOMContentLoaded', () => {
  Progress.init(); // 페이지의 모든 [data-component="progress"] 초기화
});

// 인스턴스 접근 및 setValue 호출:
const el = document.querySelector('[data-component="progress"]');
const progress = el._progressInstance;

progress.setValue(75);   // 75%로 설정 (0~100 자동 클램핑)
progress.getValue();     // 현재 값 반환
progress.increment(10);  // 현재 값 + 10
progress.decrement(5);   // 현재 값 - 5
progress.complete();     // 100%로 설정
progress.reset();        // 0%로 초기화

Sizes

.progress--sm, .progress--md, .progress--lg 클래스로 3가지 높이를 지원합니다. 크기에 따라 좁은 공간(sm), 기본(md), 강조(lg) 용도로 활용합니다.

Small (4px) 60%
Medium (8px, 기본) 60%
Large (12px) 60% 
<!-- sm: 4px (좁은 공간) -->
<div class="progress progress--sm"
  role="progressbar"
  aria-valuenow="60" aria-valuemin="0" aria-valuemax="100"
  aria-label="Small 진행률">
  <div class="progress__bar" style="--progress-value: 60;"></div>
</div>

<!-- md: 8px (기본) -->
<div class="progress progress--md"
  role="progressbar"
  aria-valuenow="60" aria-valuemin="0" aria-valuemax="100"
  aria-label="Medium 진행률">
  <div class="progress__bar" style="--progress-value: 60;"></div>
</div>

<!-- lg: 12px (강조) -->
<div class="progress progress--lg"
  role="progressbar"
  aria-valuenow="60" aria-valuemin="0" aria-valuemax="100"
  aria-label="Large 진행률">
  <div class="progress__bar" style="--progress-value: 60;"></div>
</div>
/* 크기 변형: 클래스로 높이만 변경 */
.progress--sm { height: 0.25rem; } /* 4px */
.progress--md { height: 0.5rem; }  /* 8px - 기본값과 동일 */
.progress--lg { height: 0.75rem; } /* 12px */
// 크기 변형은 CSS 클래스로만 결정
// JS에서 별도 처리 불필요
// Progress.init()이 모든 크기 변형 자동 초기화

Colors & Types

.progress--primary, .progress--success, .progress--warning, .progress--error, .progress--info 클래스로 진행 상태의 의미를 색상으로 전달합니다. 색상만으로 의미를 전달하지 않도록 라벨도 함께 제공합니다.

Primary — 기본 진행 70%
Success — 완료 100%
Warning — 저장 공간 부족 85%
Error — 업로드 실패 40%
Info — 데이터 로드 55% 
<!-- primary: 기본 진행 -->
<div class="progress progress--primary" role="progressbar"
  aria-valuenow="70" aria-valuemin="0" aria-valuemax="100"
  aria-label="기본 진행률">
  <div class="progress__bar" style="--progress-value: 70;"></div>
</div>

<!-- success: 완료/성공 -->
<div class="progress progress--success" role="progressbar"
  aria-valuenow="100" aria-valuemin="0" aria-valuemax="100"
  aria-label="완료">
  <div class="progress__bar" style="--progress-value: 100;"></div>
</div>

<!-- warning: 경고 상태 -->
<div class="progress progress--warning" role="progressbar"
  aria-valuenow="85" aria-valuemin="0" aria-valuemax="100"
  aria-label="저장 공간 부족 경고">
  <div class="progress__bar" style="--progress-value: 85;"></div>
</div>

<!-- error: 오류/실패 -->
<div class="progress progress--error" role="progressbar"
  aria-valuenow="40" aria-valuemin="0" aria-valuemax="100"
  aria-label="업로드 실패">
  <div class="progress__bar" style="--progress-value: 40;"></div>
</div>

<!-- info: 정보성 진행 -->
<div class="progress progress--info" role="progressbar"
  aria-valuenow="55" aria-valuemin="0" aria-valuemax="100"
  aria-label="데이터 로드">
  <div class="progress__bar" style="--progress-value: 55;"></div>
</div>
/* 색상 변형: .progress__bar의 background-color만 변경 */
.progress--primary .progress__bar {
  background-color: var(--color-primary);   /* #3b82f6 */
}
.progress--success .progress__bar {
  background-color: var(--color-success);   /* #10b981 */
}
.progress--warning .progress__bar {
  background-color: var(--color-warning);   /* #f59e0b */
}
.progress--error .progress__bar {
  background-color: var(--color-danger);    /* #ef4444 */
}
.progress--info .progress__bar {
  background-color: var(--color-info);      /* #3b82f6 */
}
// 색상은 CSS 클래스로만 제어
// 동적으로 색상 변경:
const el = document.querySelector('.progress');
el.classList.remove('progress--primary');
el.classList.add('progress--success');

// 진행률에 따라 색상 자동 전환:
const progress = el._progressInstance;
progress.setValue(100);
el.classList.replace('progress--primary', 'progress--success');

Disabled State & Interactive Demo

aria-disabled="true"로 비활성 상태를 표현합니다. 하단의 인터랙티브 데모에서 버튼으로 진행률을 동적으로 업데이트하는 패턴을 확인할 수 있습니다.

비활성 Progress (기본) 45%
비활성 Progress (Large) 70%
동적 진행률 업데이트 0% 

<!-- 비활성: aria-disabled="true" -->
<div class="progress-wrapper" aria-disabled="true">
  <div class="progress-header">
    <span class="progress-label" id="lbl">비활성 Progress</span>
    <span class="progress-value-text" aria-hidden="true">45%</span>
  </div>
  <div class="progress"
    role="progressbar"
    aria-valuenow="45" aria-valuemin="0" aria-valuemax="100"
    aria-labelledby="lbl"
    aria-disabled="true">
    <div class="progress__bar" style="--progress-value: 45;"></div>
  </div>
</div>

<!-- 동적 업데이트용 스크린 리더 live region -->
<p role="status" aria-live="polite" aria-atomic="true"></p>
/* 비활성 상태 */
.progress[aria-disabled="true"],
.progress-wrapper[aria-disabled="true"] .progress {
  opacity: 0.4;
  cursor: not-allowed;
}

/* 비활성 바: 회색으로 */
.progress[aria-disabled="true"] .progress__bar,
.progress-wrapper[aria-disabled="true"] .progress__bar {
  background-color: var(--color-text-secondary);
}

/* 비활성 레이블 */
.progress-wrapper[aria-disabled="true"] .progress-label,
.progress-wrapper[aria-disabled="true"] .progress-value-text {
  opacity: 0.5;
}
// 인스턴스 접근 (Progress.init() 이후):
const el = document.getElementById('progress-interactive');
const progress = el._progressInstance;

// 기본 조작:
progress.setValue(75);   // 특정 값 설정 (0~100 자동 클램핑)
progress.increment(10);  // +10% 증가
progress.decrement(5);   // -5% 감소
progress.complete();     // 100%로 완료
progress.reset();        // 0%로 초기화

// 시뮬레이션: setInterval로 자동 증가
let timer = null;
function simulate() {
  if (timer) return;
  progress.reset();
  timer = setInterval(() => {
    if (progress.getValue() >= 100) {
      clearInterval(timer);
      timer = null;
      return;
    }
    progress.increment(5);
  }, 300);
}

// live region: 완료 시 스크린 리더에 알림
const statusEl = document.querySelector('[role="status"]');
statusEl.textContent = '작업이 완료되었습니다.';
