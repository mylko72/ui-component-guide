Toast

화면 모서리에 잠깐 나타났다가 자동으로 사라지는 임시 알림 컴포넌트입니다. WAI-ARIA aria-live live region 패턴을 준수하며, info / success는 aria-live="polite", warning / error는 aria-live="assertive"를 사용합니다. data-toast-trigger 버튼으로 선언적으로 토스트를 트리거하거나, window.toast.show() API로 동적으로 생성할 수 있습니다.

Basic Toast

기본 토스트입니다. 버튼을 클릭하면 화면 우측 하단에 3초 후 자동으로 사라지는 알림이 표시됩니다. data-toast-trigger 속성만으로 선언적으로 설정하며, data-toast-message로 메시지를, data-toast-type으로 타입을 지정합니다. 

<!-- data-toast-trigger: 클릭 시 토스트 표시 -->
<button
  type="button"
  data-toast-trigger
  data-toast-type="success"
  data-toast-message="저장 완료!"
>
  Success Toast
</button>

<!-- warning / error 타입은 aria-live="assertive" (즉시 인터럽트) -->
<button
  type="button"
  data-toast-trigger
  data-toast-type="error"
  data-toast-message="오류 발생: 서버 연결 끊김"
>
  Error Toast
</button>
/* 타입별 왼쪽 보더 색상 (toast.css 내부) */
.toast--info    { border-left: 4px solid var(--color-info); }
.toast--success { border-left: 4px solid var(--color-success); }
.toast--warning { border-left: 4px solid var(--color-warning); }
.toast--error   { border-left: 4px solid var(--color-danger); }

/* 아이콘 색상 */
.toast--info    .toast__icon { color: var(--color-info); }
.toast--success .toast__icon { color: var(--color-success); }
.toast--warning .toast__icon { color: var(--color-warning); }
.toast--error   .toast__icon { color: var(--color-danger); }
// 선언적 방식: data 속성으로 설정, JS 자동 초기화
// data-toast-trigger 버튼 클릭 → toast.show() 자동 호출

// JS API 방식:
window.toast.show({
  type: 'success',
  message: '저장 완료!',
  duration: 3000,
  position: 'bottom-right'
});

// 편의 메서드:
window.toast.success('저장 완료!');
window.toast.info('정보 메시지');
window.toast.warning('주의 메시지');
window.toast.error('오류 발생!');

Position Control

data-toast-position 속성으로 토스트가 표시될 위치를 지정합니다. top-right, top-left, top-center, bottom-right, bottom-left, bottom-center 6가지 위치를 지원합니다. 

<!-- data-toast-position: 위치 지정 -->
<button type="button" data-toast-trigger
  data-toast-type="info"
  data-toast-message="상단 중앙 알림"
  data-toast-position="top-center"
>
  Top Center
</button>

<!-- 지원 위치: top-right, top-left, top-center,
               bottom-right, bottom-left, bottom-center -->
/* 위치별 컨테이너 */
.toast-container--top-right  { top: 1.5rem; right: 1.5rem; }
.toast-container--top-left   { top: 1.5rem; left: 1.5rem; }
.toast-container--top-center {
  top: 1.5rem;
  left: 50%;
  transform: translateX(-50%);
}
.toast-container--bottom-right  { bottom: 1.5rem; right: 1.5rem; }
.toast-container--bottom-left   { bottom: 1.5rem; left: 1.5rem; }
.toast-container--bottom-center {
  bottom: 1.5rem;
  left: 50%;
  transform: translateX(-50%);
}
// JS API에서 position 지정
window.toast.show({
  type: 'info',
  message: '상단 중앙 알림',
  position: 'top-center'
});

// 편의 메서드 + 옵션
window.toast.success('저장 완료!', {
  position: 'top-right',
  duration: 2000
});

Toast Types

4가지 타입으로 메시지의 심각도를 구분합니다. info / success는 role="status"와 aria-live="polite"로 현재 작업을 방해하지 않고 스크린 리더에 알립니다. warning / error는 role="alert"와 aria-live="assertive"로 즉시 인터럽트하여 긴급함을 전달합니다.

role="status" + aria-live="polite"
role="status" + aria-live="polite"
role="alert" + aria-live="assertive"
role="alert" + aria-live="assertive" 
<!-- info: 일반 정보 알림, polite -->
<button type="button" data-toast-trigger
  data-toast-type="info"
  data-toast-message="새 버전이 출시되었습니다."
>Info</button>

<!-- warning: 주의, assertive (즉시 인터럽트) -->
<button type="button" data-toast-trigger
  data-toast-type="warning"
  data-toast-message="세션이 5분 후 만료됩니다."
>Warning</button>
/* 타입별 왼쪽 보더 + 아이콘 색상 */
.toast--info    { border-left: 4px solid var(--color-info); }
.toast--success { border-left: 4px solid var(--color-success); }
.toast--warning { border-left: 4px solid var(--color-warning); }
.toast--error   { border-left: 4px solid var(--color-danger); }

.toast--info    .toast__icon { color: var(--color-info); }
.toast--success .toast__icon { color: var(--color-success); }
.toast--warning .toast__icon { color: var(--color-warning); }
.toast--error   .toast__icon { color: var(--color-danger); }
// info/success: role="status", aria-live="polite"
// warning/error: role="alert", aria-live="assertive"

window.toast.info('정보 메시지');    // polite
window.toast.success('성공 메시지'); // polite
window.toast.warning('주의 메시지'); // assertive
window.toast.error('오류 메시지');   // assertive

// show() API에서 직접 지정:
window.toast.show({
  type: 'error',
  message: '결제 처리 실패',
  duration: 5000  // 오류는 조금 더 오래 표시
});

Duration Control

data-toast-duration으로 자동 닫힘 시간(ms)을 지정합니다. 0으로 설정하면 자동 닫힘 없이 사용자가 직접 X 버튼을 눌러야 닫힙니다. 액션 버튼이 포함된 토스트는 window.toast.show() API를 사용합니다. 

<!-- duration=0: 자동 닫힘 없음 -->
<button type="button" data-toast-trigger
  data-toast-type="error"
  data-toast-message="X 버튼으로만 닫을 수 있습니다."
  data-toast-duration="0"
>수동 닫기</button>

<!-- 짧은 지속 시간 -->
<button type="button" data-toast-trigger
  data-toast-type="success"
  data-toast-message="빠르게 사라집니다."
  data-toast-duration="1500"
>1.5초 후 닫힘</button>
/* 액션 버튼 */
.toast__action {
  flex-shrink: 0;
  background: none;
  border: none;
  cursor: pointer;
  font-size: var(--font-size-sm);
  font-weight: 600;
  color: var(--color-primary);
  padding: 0.125rem 0.375rem;
  border-radius: var(--radius-sm);
  white-space: nowrap;
}

.toast__action:hover {
  background-color: color-mix(
    in srgb, var(--color-primary) 10%, transparent
  );
}
// duration=0: 수동 닫기만 가능
window.toast.show({
  type: 'error',
  message: 'X 버튼으로만 닫을 수 있습니다.',
  duration: 0
});

// 액션 버튼 포함 토스트
window.toast.show({
  type: 'info',
  message: '업데이트가 준비되었습니다.',
  duration: 0,
  actionLabel: '업데이트',
  onAction: () => {
    console.log('업데이트 실행!');
  }
});

// 반환된 ID로 특정 토스트 닫기
const id = window.toast.info('메시지');
window.toast.dismiss(id);

// 모든 토스트 닫기
window.toast.dismissAll();
